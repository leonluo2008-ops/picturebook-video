# 绘本文字保留铁律 v1+v2（2026-06-04 用户精确化 · v14 prompt 必含）

> **关键铁律**：绘本 prompt 写"保留文字"时，**必须按 v2 措辞**——不是"一直显示"，是"画面元素可自由动效"。

## 演化时间线

| 版本 | 措辞 | 实测结果 | 适用场景 |
|---|---|---|---|
| **v1 错版**（2026-06-04 上午）| "画面保持无字幕（画面无中英文文字）" | ❌ 文字被消（Good Morning clip 1） | ❌ 任何场景 |
| **v1 修复版**（2026-06-04 下午）| "保留参考图原有的所有文字元素，不重新生成任何新文字" | ✅ 文字保留但"硬贴"在画面 | ⚠️ 单图首帧模式（`--image`）|
| **v2 精确化**（2026-06-04 用户两次精确化）| "参考图原有的所有文字...作为画面元素自然融入场景，模型自行设计合理的呈现节奏与动效" | ✅ 文字保留 + 模型自由发挥（Good Morning/Good Night v2 实测通过）| ✅ 单图首帧模式（`--image`）|
| **v3 强约束**（2026-06-06 Pic Clip 2 v2 翻车驱动）| "参考图原有的所有文字（彩色字母Good afternoon + 棕色手写下午好）**必须完整保留**作为画面元素，**模型不得删除或替换**这些文字，让文字自然融入场景" | ✅ 双图 / 单图首参模式全部命中彩色字（Pic Clip 3/4/5 实测通过）| ⭐ **绘本默认（双图 + 单图首参都适用）**|

## 用户原话（两次精确化）

### 第一次精确化（v1 → v2 起点）
> "不一定是顶部彩色文字，只要参考图里有的文字，都不能移除，都视为画面元素"

### 第二次精确化（v2 完整措辞）
> "我说的保留文字，不是要求一直显示，准确的讲：是可以设计合理的运动、动效等"

## v2 措辞铁律

### ❌ 不要写（限定条件会漏字）
- "保留顶部双语"（限定顶部 = 错误，可能漏底部）
- "保留领读文字"（限定双语 = 错误，可能漏其他语言）
- "保留彩色文字"（限定彩色 = 错误，可能漏其他颜色）

### ❌ 不要写（v1 错版会消字）
- "画面保持无字幕"（v1 错版——把参考图文字也消了）
- prompt 末尾写 `no on-screen text, no subtitles`（绘本场景禁写——会把参考图文字也消了）

### ❌ 不要写（硬保留会死物感）
- 只写"保留参考图所有文字"但**不写**"作为画面元素自然融入"——模型"硬保留"产生死物感

### ✅ v2 措辞（单图首帧模式 · Good Morning/Good Night 验证通过）

```
参考图原有的所有文字（包括标题/领读文字/画面内的任何中英文）作为画面元素自然融入场景，模型自行设计合理的呈现节奏与动效（例如淡入、随镜头呼吸、配合动作出现等），不重新生成任何新文字。
```

**关键三要素**：
1. **明确范围**："所有文字"（不限顶部/双语/彩色）
2. **明确性质**："作为画面元素自然融入"（不是死物）
3. **明确动效自由度**："模型自行设计合理的呈现节奏与动效"（不写死）

### ⭐ v3 措辞（双图 / 单图首参模式 · Pic Clip 3/4/5 验证通过 · 绘本默认）

**触发场景**：`--ref-images 图1.jpg 图2.jpg`（双图合并 Clip）/ `--ref-images 单图.jpg`（单图首参模式）

**v3 跟 v2 的差异**：

| 维度 | v2 措辞 | v3 措辞（升级）|
|---|---|---|
| "保留"语气 | "作为画面元素自然融入场景" | "**必须完整保留**作为画面元素" |
| 明确禁止 | 隐含"不要重画" | "**模型不得删除或替换**这些文字" |
| 列举具体文字 | 泛指"所有文字" | "**彩色字母Good afternoon + 棕色手写下午好**"（举实际例）|
| 适用 | `--image` 单图首帧 | `--ref-images` 双图/单图首参（绘本默认）|

**v3 完整措辞**：

```
参考图原有的所有文字（<列举具体文字，例如"彩色字母Good afternoon + 棕色手写下午好"）必须完整保留作为画面元素，模型不得删除或替换这些文字，让文字自然融入场景。
```

## 绘本 vs 短剧文字处理差异

| 场景 | 文字性质 | 处理方式 |
|------|---------|---------|
| **绘本** | 文字 = 画面元素（绘制在图里）| **必须保留 + 可动效**（v2 措辞）|
| **短剧** | 文字 = 后期字幕叠加 | **可消除**（`no on-screen text, no subtitles`）|

## v2 必避反模式清单

- ❌ 任何"限定范围"的措辞（顶部/双语/彩色/某种字体）
- ❌ v1 错版措辞（"画面保持无字幕"/"画面无中英文文字"）
- ❌ prompt 末尾 `no on-screen text, no subtitles`（绘本场景）
- ❌ 只写"保留文字"不写"自然融入"
- ❌ "文字必须一直显示"（v2 反向——用户明确说**不要求一直显示**）

## v2 适用场景

- ✅ 绘本多图分镜（v14 范式）
- ✅ 绘本单图分镜（v10 范式）
- ✅ 任何绘本 prompt 写"文字"相关的描述
- ❌ 短剧/分镜生视频（用 v14 末尾 `no on-screen text` 即可）
