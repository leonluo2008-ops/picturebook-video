## 范式路由决策树（**修订版** · 移除 v7 备选）

```
Clip 涉及多张图（≥2）
   ↓
Step 1 · 场景是否连续？
   ├─ 同色系同背景（图 N + 图 N+1 = 同一场景连续动作）→ 走 v15 多图参考（默认）
   │     例：图 1（4 鸭家族）→ 图 2（单大鸭特写）= 视觉连续
   └─ **场景跳变**（水→陆 / 陆→字卡 / 主体动作组对比）→ 进入 Step 2
         ↓
Step 2 · 单段 TTS 时长？
   ├─ < 5s（短句）→ v15 多图参考仍可用（蒙太奇末帧 2s 装得下）
   └─ **> 5s（长句/家族词组/中长句）** → **必拆 Clip = 单图独立**
         例：图 4+5 跨场景 + 段 4 4.5s + 段 5 7s = 押韵连贯但场景跳变
         蒙太奇末帧 2s 装不下 7s 旁白 = 黑场 5s = 翻车
         修复 = 拆 2 Clip（图 4 单图 6s + 图 5 单图 8s）= 8s + 8s = 16s 物理上 < 15s 上限
```
