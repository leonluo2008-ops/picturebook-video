# Prompt 极简原则（信任参考图 + 关键引导而非过度约束）

> **元纠错 #9**（绘本实战沉淀）· **v2.0.0**（2026-06-14 升级）
>
> 用户原话："**视频模型可以非常好地理解参考图，如果给它参考图，提示词方面只需要做好一些关键引导就可以了，不需要描述得很详细，我们可以相信视频模型，它会自然地根据参考图结合关键指令去生成非常好的画面**"
>
> 关键指令 = 时间点 + 运镜 + 声音 + 末帧约束 + 风格词 · **不**是每个动作的精确角度 / 速度 / 度数
>
> **v2.0.0 升级**（2026-06-14 5 轮纠错沉淀）：追加 4 个新反模式（凭空捏造 / 漏字读偏 / 约束词污染 / 冲突图强行用）+ 8 项自检清单追加 4 项 · 详见 `references/prompt-overconstraint-pitfalls.md` 元教训 #10-#14

---

## 0. 核心心法（一句话）

> **prompt 是导演指令，不是动作脚本。**导演说"向前游"，不说"翅膀以 30° 角朝后展开、右脚蹼向前伸 5cm、身体前倾 15°、3.5 秒后切下一个镜头"。
>
> 模型能根据参考图自己组织细节。把 prompt 写满 = 抢了导演的工作 = 模型被迫在"听 prompt"和"看参考图"之间二选一 = 抽。

---

## 1. 必须约束（关键引导）vs 让模型自由组织

| 类别 | 必约束（关键引导）| 让模型自由组织 |
|---|---|---|
| **镜头运动** | 每镜头 1 种（推 / 拉 / 摇 / 切 / 跟）| 镜头的"呼吸感" / 速度变化 / 微抖动 |
| **末帧微动** | 必须有（不变成定格海报）| 微动的具体形态（点头 / 眨眼 / 羽毛动）|
| **声音策略** | 无 BGM / 拟声 / 不发朗读 / 末尾约束 4 词 | 拟声的音色 / 音量 / 时长 |
| **文字保留** | 顶部 1/6 锁定 · 不得删除 | 文字的呼吸/浮动节奏 |
| **风格锁** | 2D paper collage / 3D 折纸 / 油画 等 | 风格的细节笔触 |
| **大动作**（叙事驱动）| 鸭子游动 / 起飞 / 张嘴 / 排队 | 每个肢体的具体角度 / 速度 |
| **表情** | "好奇" / "开心" 等情绪锚 | 表情的微变化（眼睑开合度 / 嘴角） |
| **场景元素** | "水面波纹" / "荷叶晃动"（当叙事需要） | 波纹的具体振幅 / 荷叶倾斜角度 |

**判断口诀**：
- ✅ **写到 prompt = 模型必须执行的硬约束**（违反 = 翻车）
- ❌ **没写到 prompt = 模型自己组织的自由发挥**（让参考图说话）

## 2. 官方原话背书（4 处）

### 2.1 doc2 §1.3 · line232-237 · "分镜的 4 个组织逻辑"

> "**模型对精确时间（如 0–3 秒）的支持不稳定，强行限制时长可能导致生成结果异常**"
>
> 配套原话："**不强制限制每段时长——'优先让模型根据剧情自然生成节奏'**"

**含义**：写"0.0-3.0s 镜头一 / 3.0-8.0s 镜头二 / 8.0-13.0s 镜头三"这种按秒数硬切 = 违反官方建议 = 异常生成。

### 2.2 doc2 §2.2 · line266-269 · "动作描述要求"

> "**优先选用缓慢、轻柔、连贯的细微动作，尽量规避狂奔、大跳、剧烈翻滚等高爆发、大动态动作**"
>
> 示例：缓慢行走、轻轻抬手、微微低头、顺势坐下

**含义**：动作要轻、慢、连贯。**塞 5+ 个大动作进 1 镜头 = 高爆发叠加 = 翻车**。

### 2.3 doc2 §4 · line286-288 · "运镜写法红线"

> "**一个镜头里尽量只指定 1 种运镜方式，不要同时要求推拉摇移，会增加画面的不稳定性**"

**含义**：1 镜头 1 运镜。要换机位 = 写"镜头切至..."。**不要"推近 + 横移 + 旋转"叠**。

### 2.4 doc2 §6 · line320-389 · 实战案例范本

**官方案本 1**（line 198-211 · 宿舍情感短剧）：

```
镜头 1：傍晚时分，女孩脚步轻快地走到宿舍门口，镜头中景平稳跟拍，
暖黄色日光从窗外洒进走廊，她在门口停顿一下，深呼吸，表情略带紧张。
```

**官方案本 2**（line 244-253 · 悬崖对手戏）：

```
镜头 1：傍晚，镜头从红衣女子侧面中景缓慢推进，她站在悬崖边拿起酒壶喝酒，
衣袂在山风中轻轻摆动，镜头环绕她半圈，从正面推到背影，远处隐约可见竹林中的黑衣人影。
```

**官方 2 个范本特征**：
- ✅ 写"脚步声 / 衣袂摆动 / 远处隐约"——**用环境元素暗示动作**
- ✅ 写"表情略带紧张 / 目光坚毅"——**情绪外化为可观察细节**
- ❌ **没有**"翅膀朝后 30° / 脚蹼前伸 5cm / 表情变化 0.3s"这种精确量化
- ❌ **没有**"0-3s 镜头一 / 3-8s 镜头二 / 8-13s 镜头三"这种秒数切分

## 3. 5 个反模式（绘本 prompt 实战踩坑）

### 反模式 1 · 按秒数硬切分镜

**症状**（绘本 clip1 v1 翻车）：prompt 写"镜头一（0.0-3.0s · opening）/ 镜头二（3.0-8.0s · middle）/ 镜头三（8.0-13.0s · final）"= 触发官方 §1.3 红线 = 模型在切点抽搐。

**修复**：删秒数，只写"镜头一 / 镜头二 / 镜头三"。**让模型自己分配每镜时长**。

**官方反例**：doc2 §1.3 官方原话"强行限制时长可能导致生成结果异常"。

### 反模式 2 · 单镜头塞 4+ 动作

**症状**（绘本 clip1 镜头三）：prompt 写"yellow duck **swims forward** to the right, **wings spread**, **webbed feet paddle**, **water splashes trailing**"= 1 镜头 4 动作 = 高爆发叠加 = 模型二选一时抽。

**修复**：每镜头 1-2 核心动作。其他让参考图说话：

```
# ❌ 错
镜头 3：鸭子向右游，翅膀张开，脚蹼划水，身后拖出白色水花，镜头锁在最后画面。
末帧：鸭子继续划水，水波不停，翅膀轻轻扇动。

# ✅ 对
镜头 3：镜头中景平稳跟拍，鸭子一家继续向前游动，背景的水面波纹和撕纸叶片微微晃动，自然过渡到画面收尾。<几声轻柔的水花声>。
```

### 反模式 3 · 动作语义跟参考图姿态冲突

**症状**（绘本 clip1 镜头三）：prompt 写"swims"（游泳姿态）+ 参考图 3 实际是"冲浪 / 跑步 / 滑翔"姿态（翅膀朝后张开、橙脚朝前伸、身后有浪花速度线）= 语义冲突 = 模型困惑。

**修复**：
- **A. 让 prompt 跟参考图姿态一致**——"glides on a wave like a surfer, wings spread as if catching wind, webbed feet tucked under, no paddling motion"
- **B. 删具体动词，让模型照参考图**——"鸭子一家继续向前移动" + 让模型自己决定是 swim / glide / surf
- **官方反例**：doc2 §1 注意"避免语义冲突（如同一主体出现矛盾特征）"。

### 反模式 4 · 自相矛盾的末帧约束

**症状**（绘本 clip1 末帧）：prompt 写"camera **locks in final frame**"（锁住 = 静止）+ 后面再写"末帧必须包含至少 1 个动作元素"（必须动）= **自相矛盾** = 模型要么锁死要么狂动。

**修复**：只写一种。绘本场景 = **必动**（避免定格海报），所以：

```
# ❌ 错
camera locks in final frame。
末帧必须包含至少 1 个动作元素（the yellow duck keeps gently paddling forward...）。

# ✅ 对
镜头 N 末尾：鸭子继续向前移动，水波和叶片微微晃动，自然过渡。
```

### 反模式 5 · 风格词只写 1 次（防风格漂移不够）

**症状**：风格词（"2D paper collage / 3D 折纸 / 油画"等）只在段 3 写 1 次 = 模型在长 prompt 中可能"忘了" = 漂移到 3D / 写实。

**修复**：
- 段 1 主体定义里嵌风格锚（"yellow duck family in 3D paper-collage low-poly texture"）
- 段 3 单独写一段风格锁
- **不**在每镜头都重复（避免啰嗦）—— 写在关键锚点就够

**官方反例**：doc2 §533-549 "风格漂移"+"加入明确的风格约束词"。

### 反模式 6（**v2.0.0 新增**）· 凭空捏造参考图没有的元素（绘本 v1/v2 翻车）

**症状**：写 prompt 前**没**用 vision 看参考图 = 凭印象 + 抄旧 prompt = 编出参考图里**根本不存在**的元素（绘本 v1 编了"4 鸭+荷叶+歪头"，实际 3 张图都是 1 只鸭 + 没有荷叶 + 鸭子没歪头）。

**修复**：
- **Step 1**：vision 全 N 张图（不抽样）—— 已沉淀进 SKILL.md Step 2.0
- **Step 2**：列"图里有/没有"清单（写到 image-inventory.md）—— "没有"列 = 警告
- **Step 3**：prompt 每个名词必须在"有"列里
- 详见 `references/prompt-overconstraint-pitfalls.md` 元教训 #11

**判定**：
```python
inventory = {
    "图1": {"有": ["1 只黄鸭", "草地", "纸艺"], "没有": ["小鸭", "荷叶", "歪头", "水花"]},
    "图2": {"有": ["1 只黄鸭", "水面静态蓝圈波纹"], "没有": ["小鸭", "荷叶", "水花"]},
    "图3": {"有": ["1 只黄鸭", "踏浪姿态", "蓝色水花"], "没有": ["小鸭", "荷叶", "游泳"]},
}
# ❌ 错：4 鸭立荷叶歪头 = 4 鸭/荷叶/歪头 都不在"有"列
# ✅ 对：1 只黄鸭站草地 = "1 只黄鸭"/"草地" 都在"有"列
```

### 反模式 7（**v2.0.0 新增**）· 漏字读偏官方原话（绘本 v3 翻车）

**症状**：引用官方"**1 种运镜方式**"漏"**运镜**"4 个字 → 读成"**1 动作**" → 给用户推"1 镜只 1 动作" → 违反官方原意（"1 镜多动作 OK"）。

**修复**：
- **凡引用官方原话 = 必 grep 原文档验证 + 标注 4 维属性**（原文+位置+语气+上下文）
- 漏 1 字 = 错全意 · 缩 1 句 = 断章取义 · 凭记忆 = 翻车
- 详见 `references/prompt-overconstraint-pitfalls.md` 元教训 #12

**反例 vs 正例**：
```markdown
# ❌ 错：凭缩略记忆复述
"官方说 1 镜 1 动作"

# ✅ 对：原文 + 4 维属性
doc2 §4 line286-288（红框·"注意"语气）："**一个镜头里尽量只指定 1 种运镜方式**"——仅指运镜（推/拉/摇/移/切），动作可以多
```

### 反模式 8（**v2.0.0 新增**）· 泛泛堆"不要 X"约束词（绘本 v4 翻车）

**症状**：末尾约束写 6 个"无 X" + 2 个"音效只保留" + 1 个"全程无 BGM" = 泛泛"不要 X"堆砌 = 污染模型。

**修复**：
- **约束词只对"已知出错元素"用**（字幕/Logo/水印 3 类官方原话）
- 4 词末尾约束（无人声/无歌唱/无配音/无朗读）是绘本专用 = **已确认保留**，**不**开问卷
- 详见 `references/prompt-overconstraint-pitfalls.md` 元教训 #14

**反例 vs 正例**：
```python
# ❌ 错：泛泛堆砌
保持无字幕、无水印、无 Logo，无人声、无歌唱、无配音、无朗读。
音效只保留环境细节：水花声、纸页翻动等短促物理声。
全程无背景音乐。

# ✅ 对：精简到官方 3 类 + 4 词（4 行）
保持无字幕。
不要生成水印。
不要生成 Logo。
无人声、无歌唱、无配音、无朗读。
```

### 反模式 9（**v2.0.0 新增**）· 冲突参考图强行用（绘本 v3 翻车）

**症状**：参考图 N 跟主题冲突（@Image3 踏浪跑步姿态 vs 主题"鸭子游泳"）= 强行写 prompt 让模型"游"= 末帧仍是踏浪 = 修 prompt 修不好。

**修复**：
- **跳过冲突图 = prompt 里不引用 + ref_images 也不传 + 复用 @ImageM + 描述"自然延伸"**
- 写法关键："参考 @ImageM 的同一只鸭子，镜头继续中景跟拍，鸭子从浮水状态开始向画面右侧游动"
- "3 镜头 = 3 张图"是死规则**已废**——可以复用参考图
- 详见 `references/prompt-overconstraint-pitfalls.md` 元教训 #13

## 4. 重写后的 绘本 clip1 范例（官方原话风格的"信任参考图"版本）

```
@Image1 @Image2 @Image3 as the visual reference. The yellow duck family in a paper-collage children's picture book style, soft warm lighting, gentle and cheerful mood.

镜头 1：镜头从鸭子一家平视的远景缓慢推近，4 只鸭子站在绿色荷叶上，大鸭子轻轻点头，水面泛起几圈细小涟漪。<一声轻微的水花声>。
镜头 2：镜头切到中景，聚焦大黄鸭，大鸭子轻轻歪头看向镜头，翅膀微微张开，3 只小鸭在身边跟随。整段保持连贯，一气呵成。
镜头 3：镜头中景平稳跟拍，鸭子一家继续向前游动，背景的水面波纹和撕纸叶片微微晃动，自然过渡到画面收尾。<几声轻柔的水花声>。

整体保持 2D/3D 纸艺拼贴风格（paper-collage children's picture book style），与参考图高度一致；
原图顶部 1/6 区域的彩色英文字母与红色中文字全程保留在画面中，不被删除或替换。
保持无字幕、无水印、无 Logo，无人声、无歌唱、无配音、无朗读。
音效只保留环境细节：水花声、纸页翻动等短促物理声。
全程无背景音乐。
```

**关键改动**（5 条全命中反模式 1-5）：
1. **去掉所有秒数**（0.0-3.0s / 3.0-8.0s / 8.0-13.0s 全删）
2. **每镜头 1 种运镜**（缓慢推近 / 切到中景 / 平稳跟拍）
3. **每镜头 1-2 核心动作**（点头 / 歪头 / 继续游动）
4. **不写"swims vs surfing"矛盾动词**——让参考图说话
5. **末帧 = 让模型自然过渡**（不再写"必须包含至少 1 个动作元素"自相矛盾）

## 5. 自检清单（写完 prompt 必跑）

| # | 检查 | 期望 |
|---|---|---|
| 1 | prompt 里**没有**精确秒数（"0-3s / 3-8s / 8-13s"）| ✅ 全删 |
| 2 | 每个镜头描述**只有 1 种运镜** | ✅ 推 / 拉 / 摇 / 切 / 跟 5 选 1 |
| 3 | 每个镜头描述**只有 1-2 个核心动作** | ✅ "继续游动" 而不是 4 动作堆叠 |
| 4 | prompt **没有跟参考图姿态矛盾**的动词 | ✅ "swim" / "run" / "fly" 选 1 个 · 或删动词让模型决定 |
| 5 | 末帧约束**不自相矛盾**（不"锁" + 必"动"）| ✅ 只写一种 |
| 6 | 风格词**至少 2 处锚定**（段 1 主体定义 + 段 3 风格锁）| ✅ 漂移防御 |
| 7 | 末尾约束 4 词**齐全**（无人声/无歌唱/无配音/无朗读）| ✅ 防莫名说话声 |
| 8 | 镜头描述**用环境元素暗示动作**（水面 / 风 / 叶片晃动）| ✅ 仿官方案本 1-2 写法 |
| 9（**v2.0.0 新增**）| prompt 里**每个**名词都在 image-inventory "有"列里？| ✅ 不凭空捏造 |
| 10（**v2.0.0 新增**）| 引用官方原话时**必带 4 维属性**（原文+位置+语气+上下文）？| ✅ 不漏字不断章 |
| 11（**v2.0.0 新增**）| 末尾约束 ≤ 4 行（"无字幕 / 不要水印 / 不要 Logo / 4 词末尾"）？| ✅ 不泛泛堆"不要 X" |
| 12（**v2.0.0 新增**）| prompt 引用的 @ImageN 都跟主题不冲突？| ✅ 跳过冲突参考图 |

**12 项全过 = prompt 进入提交阶段。任 1 项失败 = 修复后重跑**。
