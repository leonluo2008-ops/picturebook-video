## 范式核心

**v15 = v14 骨架（4 段 prompt）+ 3 项 Say 绘本优化**

| 维度 | v14 | **v15** |
|---|---|---|
| 段 1 主体定义 | ✅ 4 要素 | ✅ 4 要素（继承）|
| 段 2 分镜绑定 | ✅ @图N + v2 文字保留 + 收势 | ✅ + **运镜调性**（6 必问 #4 决定）|
| 段 3 风格锁定 | ✅ `2D paper collage style` | ✅（继承）|

| 运镜三禁 | ❌ | ❌ **撤回**（"3 秒后砰弹出"等秒级控制 = 优势）|

## 跟 v14 的 3 个差异

### 差异 1 · 启动前 6 必问（v14 没有）

**v14**：启动即用模板，不要求先问用户
**v15**：启动前必问 6 件事（**运镜调性**是新增维度）

```
1. 视频比例（默认 16:9）
2. 目标时长（领读型 = 单 Clip 4s）
3. 切分方案（领读型默认 = 4-Clip 合并）
4. **运镜调性**（活泼有动感 / 治愈舒缓 / 动态开合 / 静态展示）← 新增
5. 运镜规则（按画面类型定制 / 套公式）
6. 范式风格（v10 领读型 / v8 调性 / v7 静默 / v9 整 Clip）
```

**为什么加**：本次踩坑最大问题就是"运镜调性不符"——AI 默认给平淡的运镜，缺乏活泼感。

### 差异 2 · 撤回"运镜三禁"（v14 反向）

**v14 写法**："1.5 秒""3 秒后"等秒级控制 —— 合法（v10 范式核心）
**v15 不再禁时间感描述 + 秒级控制**——**撤回"运镜三禁"**

**真正必避的只剩 1 条**："套运镜公式"（"中景→中近景缓推"模板套所有 Clip）

### 差异 3 · v7-Say 密集音效型（新子范式）

**v14 v7 静默型**（段 4 写法）：
```
No background music, no human voice, no narration, no singing;
```
- 音效稀疏（2-3 个/8s）
- 拟声模糊（"伴随环境音"）

**v15 v7-Say 密集音效型**（段 4 写法）：
```
无任何背景音乐、无旁白人声、无哼唱。
```
- 音效密集（5 个/8s）
- **拟声+动作一对一**（每个镜头配 1 个具体拟声词）
- **拟声词具体化**（咚/呼呼/砰/叮/沙沙，5 种）
- **收尾双拟声组合**（短收势 + 收尾"叮"+"沙沙"）
- **双重禁 BGM**（段尾明写 + 段内不写"持续铺底"）

## v15 完整 4 段结构

### 段 1 · 主体定义（继承 v14）

```
一只可爱的卡通小熊，由深浅棕色卡纸拼贴而成，圆头圆耳，黑色圆眼睛，浅色口鼻部；
画面上方居中是彩色拼贴字母"说Say!"（S 红、a 黄、y 绿，"说"字金黄色）；
四周散落彩色音符和白色对话气泡；背景为浅蓝渐变天空 + 鲜绿草地；
整体是儿童绘本封面节奏。
```

### 段 2 · 分镜绑定 + 文字保留 v2 + 运镜调性（v15 升级）

```
@图1 作为唯一参考帧；

镜头一：小熊从画面下方外快速跳入到中央，双脚落地"咚"一声；
镜头二：双手快速举到脸颊两侧，嘴巴同步张大呈 O 形，"呼呼"纸张被风卷动；
镜头三：镜头快速从中景推进到脸部特写，文字"说Say!"同步从两侧"砰"弹出到画面上方居中，"叮"清脆一声；
镜头四：镜头缓推持续，文字在画面中自然呼吸，纸张摩擦"沙沙"；
镜头五（收势）：画面最终定格在小熊双手捂嘴呼喊 + 文字饱满呈现的姿态，镜头完全静止不动，不渐隐不淡出，停留至最后一帧，配合一声清脆的"叮"+轻柔的纸张摩擦"沙沙"作为收尾音效；

参考图原有的所有文字（中文"说"、英文"Say"、所有装饰文字）作为画面元素自然融入场景，
模型自行设计合理的呈现节奏与动效（例如淡入、随镜头呼吸、配合动作出现或消失等），
不重新生成任何新文字；
```

**v15 段 2 特点**：
- **运镜调性**（活泼有动感）= 5 镜头结构（≤ 5 镜头允许，时间公式允许）
- **秒级控制**（"快速跳入""同步弹出"）—— 跟 v10 范式一致
- **5 拟声+动作一对一**（咚/呼呼/砰/叮/沙沙）
- **v2 文字保留措辞**（自然融入，可消失）

### 段 3 · 风格锁定（继承 v14）

```
2D paper collage style, 儿童绘本纸艺拼贴风，剪纸 + 撕纸 + 卡纸拼贴质感，
边缘可见纸张纤维，色彩饱和明亮，童趣活泼。
```

### 段 4 · BGM 段（v15 新增 v7-Say 密集音效型）

```
无任何背景音乐、无旁白人声、无哼唱。
```

**v15 段 4 三选项**：
- **A. v7 静默型**（安静/治愈）= `No background music, ...`
- **B. v8 调性匹配型**（多情绪）= 删 `No background music` 让 Seedance 配
- **C. v7-Say 密集音效型**（**v15 新增**·活泼/领读型）= `无任何背景音乐、无旁白人声、无哼唱。` + 5 拟声/8s

## v15 必跑启动前 6 必问

```
[ ] 1. 视频比例（默认 16:9）
[ ] 2. 目标时长（领读型 = 单 Clip 4s）
[ ] 3. 切分方案（领读型默认 = 4-Clip 合并）
[ ] 4. **运镜调性**（活泼有动感 / 治愈舒缓 / 动态开合 / 静态展示）← v15 新增
[ ] 5. 运镜规则（按画面类型定制 / 不套"中景→中近景缓推"公式）
[ ] 6. 范式风格（v10 领读型 / v8 调性 / v7 静默 / v9 整 Clip）
```

## v15 必避反模式

- ❌ 启动前不跑 6 必问（直接套模板 = 可能跑偏）
- ❌ 套运镜公式（"中景→中近景缓推"模板套所有 Clip）
- ❌ 写 "[Sound effect: ...]" 独立块（句号把视觉切成多段）
- ❌ 收势词后追加内容（覆盖收势指令）
- ❌ 视觉段写 "no speech / no narration / no BGM" 否定句（v14 三铁律）

**撤回的反模式**（之前以为错其实合法）：
- ✅ 允许"3 秒后""砰弹出"等秒级控制（v10 范式核心）
- ✅ 允许"5+ 镜头"结构（分镜时序窗用 `from 0.0s to 4.0s @Image1`）
- ✅ 允许时间感描述（"镜头快速推进""急促的纸张摩擦"）
