# v3 · 叙述法过渡

**测试时间**：2026-06-02 上午
**绘本**：Cactus 仙人掌
**结果**：❌ 失败

## 范式核心
prompt 末段写"transitions seamlessly to @Image2 as the second half"等叙述法，让模型自己决定切换时机。

## 失败原因
1. **切换不准**：写"在 1.2s 切换"，模型实际在 3s 才切
2. **标题被吃**：2.jpg 的"A Green Cactus"标题文字被模型当字幕重渲染丢失
3. **绘本翻页感**：模型从头渲染场景，连接像 GIF 随机播放

## 关键教训
- 视频模型**不擅长精确时间控制**（"X 秒切换"不可靠）
- 视频模型**不擅长保留图内文字**（文字会被重渲染）
- 叙述法 / 自然语言写法**精度低**——必须用结构化时间窗
