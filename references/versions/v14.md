## 范式核心（4 段式 prompt 公式）
1. **段 1 主体定义 + 绑定**：
   ```
   将图片1中的 [主体] 定义为主角@图片1
   将图片2中的 [主体] 定义为 [角色2]@图片2
   ```
2. **段 2 风格锚定**：
   ```
   2D paper collage illustration with hand-cut paper texture, warm cozy palette, soft watercolor edges, childlike handmade feel.
   ```
3. **段 3 分镜时序 + 元素绑定**：
   ```
   Shot 1: from 0.0s to 4.0s @Image1 in this shot, [图1元素] [动作];
   Shot 2: from 4.0s to 8.0s @Image2 in this shot, [图2元素] [动作];
   ```
4. **段 4 BGM 段**（A/B 模式二选一）：
   - **A 模式（用户传 BGM）**：`参考音频1中的 [乐器] 作为背景音乐，在整个视频中以相同的调性播放`
   - **B 模式（无 BGM）**：`全程保持 BGM 静默，仅生成环境音效（鸟叫/风声等拟声）`

## A/B 模式决策表
- **A 模式**：用户上传 mp3 BGM → 必传 `--audio` + prompt BGM 段
- **B 模式**：用户只传图 → 删 `--audio` + prompt BGM 段改"仅环境音效"
- **共同点**：音效必有 / 人声无 / 字幕水印无

- **6 条铁律**（在 SKILL.md §6）：
