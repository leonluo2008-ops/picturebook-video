```
BGM 段：same warm xylophone and pizzicato melody from the previous clip continues throughout this entire clip. The BPM stays gentle at 80bpm, the chord progression stays in C major, the tempo holds steady.
```

## 关键句式（v10 核心）
- 段 1：分镜引导句（"This is a storyboard..."）
- 段 2-3：时序窗 + 动作（v5/v6 风格）
- 段 4：收势词
- 段 5：**BGM 段（必含 same ... from the previous clip continues）**
- 段 6：音频禁令（v7 风格）
- 段 7：风格锚点

## 关键铁律
- **v10 收势位置**：只有最后 1 个 clip（clip N）有 `single quiet warm chime as the final note` 收势，前 N-1 个全部 `BGM continues softly into the next moment`
- **风格锚点统一**：所有 clip 用同一句风格描述

## 适用场景
