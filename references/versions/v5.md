# v5 · 显式分镜时序

**测试时间**：2026-06-02 下午
**绘本**：Cactus 仙人掌
**结果**：✅ 切点里程碑

## 范式核心
v4 升级：把切换时间显式写进 prompt，结构化时序窗：
```
from 0.0s to 1.2s @Image1 is the opening shot, in this shot {动作};
from 1.2s to 8.0s @Image2 is the second shot, in this shot {动作};
```

## 解决了什么
- 切换点 100% 精准（1.2s 切到图 2，验证）
- 提示词结构清晰，可读性高

## 用户金句
"分镜时序的方式是可行的"

## 关键经验
- **结构化时间窗 > 自然语言时间描述**（"switch at 1.2s" 不可靠）
- 配合 `holds steady for the rest of this shot` 可控尾段
- 用户原话："能让画面在目标时段精准切换并静止，那么也可以让画面在目标时段动起来"——这启发了 v6

## 状态
