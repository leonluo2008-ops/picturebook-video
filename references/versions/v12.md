```

```

## 关键教训
1. **多图分镜 prompt 写主体描述会覆盖参考图**——必须用"主体定义 + @图片N 绑定"显式声明（v14 修复）
2. **音频 URL 必须公网可达**——飞书云盘 URL Seedance 不可达（需登录态）
3. **音频限制**：单音频 [2,15]s，最多 3 个，总时长 ≤15s
4. **Seedance 默认 audio role 是音色**，必须显式 `role: reference_audio`
