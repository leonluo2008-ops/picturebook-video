## 6 个具体坑（沉淀到 SKILL.md #25-#31）

### 坑 1：旁白语言版本理解错误（→ #25）

- **用户原话**："我只要中文版旁白"
- **我的错版理解**："画面只留中文'说'，删英文 Say"
- **正确理解**（PIG 双栏表 + Say 双栏表交叉验证）：
  - 素材表 = 双栏（英文版 + 中文版）
  - 中文版格式 = `<中文主旁白> + <小写英文跟读>`
  - 画面文字 = 中文版**完整那一行**（不是只中文，不是英文版）
- **PIG/Say 句式对照**：
  - 封面：英文版 `PIG!` / 中文版 `小猪 PIG！`
  - 故事：英文版 `A pink pig.` / 中文版 `粉色的小猪，a pink pig`
- **诊断口诀**："用户说 X 版" = "X 版**完整那一行**" = 不是"删 Y 留 X"

### 坑 2：彩色文字"自然融入"被自己报为 bug（→ #26）

- **用户原话**："彩色文字不需要一直持续出现在画面中，可以根据情节消失"
- **v1 错版行为**：vision_analyze 抽帧报"末帧 Say 字母丢失" → 把它当 bug → 启动 v2 修复
- **正确理解**：v2 措辞"模型自行设计合理的呈现节奏与动效"鼓励文字淡入淡出/末帧渐隐
- **新规**：
  - ❌ 单测门不再报"末帧文字消失"为 bug
  - ✅ 硬约束：参考图所有文字**必须至少出现一次**（完全丢失才是 bug）
  - ✅ 淡入淡出 / 随镜头呼吸 / 配合动作出现或消失 = 全部合规

### 坑 3：运镜套固定公式（→ #27）

- **用户原话**："刚才生成视频时的运镜根本不符合领读绘本，运镜要服务于故事，不是固定公式"
- **v1 错版 prompt**：
  ```
  起：小熊双手捂嘴定格 1.5 秒
  承：镜头缓推 3 秒
  转：小熊表情强化
  合：画面最终定格...
  ```
- **错版问题**：套"起承转合"叙事弧 + 固定"中景→中近景缓推 3 秒"模板到所有 Clip
- **正确做法**：按画面类型定制
  - 封面 → 缓推（建立感）
  - 角色动作 → 静态或极缓（突出动作）
  - 多人场景 → 拉远 or 水平 pan
  - 情感场景 → 几乎不动
  - 礼物/递出 → 缓推 + 微向下倾
  - 彩带/挥手/风动 → 静态（让物体自己动）

### 坑 4：4s 反馈被擅自套用到所有 Clip（→ #28）

- **用户原话**（15:51）："效果是可以的，只是时间太长了，这个视频 4s 时长完全足够了。"
- **我犯的错**：把"4s"套用到 Clip 2-10
- **正确做法**：4s 是**硬下限**（`4s ≤ duration ≤ 15s`），不是推荐值
- **判定**：用户说"4s 完全足够"= **只针对那个 Clip**，**不是说所有 Clip 都 4s**

### 坑 5：猜"4s 装 2 段旁白"= 凭直觉不查 skill（→ #29）

- **用户原话**（15:54）："你TM别乱猜，你好好看skill中clip的计时规则"
- **我犯的错**：
  - 用户说"Clip 1 砍到 4s" → 我擅自把 Clip 2-10 也按 4s 处理
  - 用户问"4s 装 2 段旁白" → 我**没看 skill**直接猜"每段 2s"
  - 直接说"我猜" → 用户爆粗
- **正确做法**：
  - **不擅自延伸**——用户说 A，只做 A
  - **不凭直觉猜**——拿不准时**先看 skill**（Step 2 分镜计时规则 / 0.3s/字 / 4s 硬下限 / 合并判断条件）
  - **不要直接说"我猜"**——猜也要说"按 skill 规则推导是 X"
