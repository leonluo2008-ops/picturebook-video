## 🛠️ 检测方法（撞天花板必走 3 步）

### 步骤 1：检查 test-prompts 是否引用已废编号

```bash
# 已废编号 = # 编号格式 = v1.2.0+pic22 起禁用（改用主题编号）
grep -E "铁律 #[0-9]+|#[0-9]+ 铁律|铁律第[一二三四五六七八九十]+条" test-prompts.json
# 命中 = test-prompts 引用了已废编号 = 必重写
```

**反例**（v1.2.0+pic22 撞 100% 时）：
```json
{
  "id": 5,
  "prompt": "...",
  "expected_should_trigger": "应命中 v15director §1.3 时长档位 + 总镜头数 ≤ 时长 ÷ 1.5s + **不**命中已废的'铁律 #32 #33 旁白朗读公式'（# 编号已删）",
  ...
}
```

**问题**：`铁律 #32 #33` 字符串在 SKILL.md 文本里**确实**存在（出现 在历史变更说明里）= 脚本匹配命中 = **虚假命中**。

### 步骤 2：检查 test-prompts 引用文件是否真实存在

```bash
# 抽所有 markdown 文件引用
python3 << 'EOF'
import json, re, os
with open('test-prompts.json') as f:
    prompts = json.load(f)

skill_dir = '/home/luo/.hermes/profiles/huiben/skills/creative/picturebook-video/'
all_text = json.dumps(prompts, ensure_ascii=False)

# 抽 references/*.md 引用
refs = re.findall(r'references/([\u4e00-\u9fff\w-]+\.md)', all_text)
unique_refs = sorted(set(refs))

print("=== 引用文件存在性检查 ===")
for ref in unique_refs:
    fp = os.path.join(skill_dir, 'references', ref)
    exists = os.path.exists(fp)
    print(f"  {'✅' if exists else '❌'} {ref}")
EOF
```

**反例**（v1.2.0+pic22 撞 100% 时）：
- `references/旁白朗读时长计算.md`（**不存在** · 我之前以为存在 · 实际从未创建）
- `references/2026-06-07-pic4-no-v5-rhythm-formula.md`（已删 · 引用了失效文件）

## 🎯 元教训（4 问审查 = 铁律级）

### 元教训：达尔文撞天花板必查 3 步（v1.2.0+pic22 幻觉天花板教训）

**违反 = 虚假命中 = 达尔文失去价值**（v1.2.0+pic22 撞 100% 撞成幻觉天花板 = 浪费时间）

- ❌ 看到 100% 就停止 hill-climbing（**没查真实覆盖率** = 幻觉天花板）
- ✅ 撞天花板后必走 3 步：
  1. 检查 test-prompts 引用已废 # 编号（grep "铁律 #")
  2. 检查 test-prompts 引用文件真实存在（Python 检查）
  3. 全文搜真实覆盖率（不依赖脚本匹配）
- ✅ 真实覆盖率 ≥ 80% + 单条铁律都有 ≥1 个 test-prompt 覆盖 = 真撞
- ✅ 真实覆盖率 < 30% = 幻觉天花板 = 必重写 test-prompts

### 触发场景

任何达尔文循环撞天花板（任何 skill 8 维评分 / 核心任务支持率 = 100%）后必走本检测。

### 4 问审查

1. **违反 = 必然翻车吗？**（不是必然翻车 · 是浪费达尔文循环时间）
2. **能合并到其他铁律吗？**（能 · 合并到 `darwin-skill` 里 = 全 skill 通用）
3. **能降级到 references 或 SKILL.md 正文吗？**（**降级到本文件** = 不升 picturebook-video 铁律表 = 是达尔文方法论补丁）
4. **有官方文档原话支持吗？**（**无** = 用户方法论沉淀 = 元教训级）

**结论**：**不**升 picturebook-video 铁律表（**是达尔文方法论补丁**），但**必须**写入 darwin-skill / picturebook-video references 作为方法论沉淀。
