## 决策记录

### v1.0.0 重构决策

1. **v15 框架保留不动**——只从 v13 借"@Image 显式绑定"
2. **每个镜头用哪个图 = 该镜头展示什么**
3. **不动"v15.1 标版"作为范式名**——保留 6 处"标版"字样（v15.1 范式名 / 章节引用 / 案例标注）
4. **末帧 ≠ 定格海报**——4-5s 是画面继续微动
5. **参考图视觉权重高 = feature**——文字保留写"参考图作为视觉基线"
6. **MVP 选 B 方案**——补铁律 #36 措辞升级
7. **git 流程**：main 全部 commit → push → 从 main 最新开优化分支
8. **图片分析冲突时**——直接重看图（vision_analyze 第二次复核）

### v1.0.0 实战决策

1. **不开 PR**（用户原话铁律 #49）——重构完成 + evals 100% ≠ 实战可用
2. **Pic2 Clip 1 节奏可用**——接受翻车只修 C（铁律 #42 接受现状）
3. **Cat v15 范式精准度迁移** = C 子 agent 必填 clip_narrative + 1.5s 镜头一
4. **D 子 agent 不擅自改 prompt**（铁律 #45）——翻车报回主 agent 决策
5. **视频交付不抽帧**（铁律 #29 强化）——D 不发飞书 + 主 agent 不主动抽帧
