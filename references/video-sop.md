# video-sop - 即梦（Dreamina）视频创作标准工作流程

> 来源：云盘 jimeng-2026-05-26-2894-video-sop.md
> 即梦官方9阶段SOP，涵盖从创意发散到最终成片的完整流程。

---

## 技能描述

即梦（Dreamina）视频创作标准工作流程，涵盖从创意发散到最终成片的9个完整阶段，支持**故事成片**（故事短片、动画故事、小说漫改、情感短片、电影感短片）和**营销广告**（产品广告、带货视频、品牌宣传、营销短片）两大类内容创作。

本SOP是所有视频创作的主流程框架，必须严格遵循阶段顺序执行，每个阶段完成后进入下一阶段，不允许跳步。

---

## 创作整体流程（9个阶段）

```
Phase 0: 创意发散 → 
Phase 1: 内容确认 → 
Phase 2: 内容大纲 → 
Phase 3: 剧本创作 → 
Phase 4: 素材挖掘 → 
Phase 5: 分镜设计 → 
  ↳ Step 1 分镜切分 (script-chunk)
  ↳ Step 2 分镜计时 (shots-timing)
  ↳ Step 3 分镜组合 (shots-assembly)
  ↳ Step 4 连贯性校验 (scene-reflection)
Phase 6: 需求确认 → 
Phase 7: 参考素材生成 → 
Phase 8: 分镜生视频 → 
Phase 9: 视频剪辑/成片 → 
```

---

## 各阶段详细说明

### Phase 0：创意发散
**适用场景**：用户只有一个模糊想法/主题/一句话灵感，没有具体内容
**调用技能**：
- 故事成片 → `story-idea`（故事短片大纲创作）
- 营销广告 → `ecom-idea`（营销广告创意构思）

### Phase 1：内容确认
**核心工作**：和用户确认目标总时长、宽高比、整体风格、其他特殊要求
**输出**：《创作需求确认表》

### Phase 2：内容大纲
**调用技能**：story-idea / ecom-idea
**输出**：故事概览、角色列表、三幕式结构、主题/卖点

### Phase 3：剧本创作
**调用技能**：story-script / ecom-script
**输出**：完整可拍摄剧本（场景+动作+对话）

### Phase 4：素材挖掘
**调用技能**：ref-extract
**输出**：结构化素材清单（角色+道具+场景，标注主体来源）

### Phase 5：分镜设计（4步走）

| Step | 技能 | 工作 |
|------|------|------|
| Step 1 | script-chunk | 将剧本拆分为单个分镜 |
| Step 2 | shots-timing | 计算每镜时长，汇总总时长 |
| Step 3 | shots-assembly | 合并为4-15秒的clip |
| Step 4 | scene-reflection | 校验一致性，修复问题 |

### Phase 8：分镜生视频
**调用技能**：video-prompt
**核心工作**：将每个clip翻译成AI可执行的prompt，并行生成所有视频片段

### Phase 9：视频剪辑/成片
**核心工作**：ffmpeg拼接所有clip，智能生成背景音乐，输出最终成片

## 核心规则

1. **不跳步原则**：必须按阶段顺序执行，每个阶段需要用户确认后再进入下一阶段
2. **并行生成原则**：同一阶段多个生成任务必须一次性并行提交，不串行分批
3. **时长硬约束**：每个AI生成的Clip必须在4-15秒之间，绝对不能超限
4. **一致性原则**：参考主体生成后，所有后续生成必须保持一致性
5. **最小修改原则**：用户反馈问题时，只修改有问题的部分，不重新生成全部

---

## 单Clip直达规则（例外情况）

当且仅当同时满足：
1. 用户目标是生成单个Clip（单个4-15秒视频片段）
2. 需求已经足够明确，可直接生成

此时：
- 纯文字 → 直接 `text2video`
- 已有首帧图 → 直接 `image2video`
- 多参考素材 → 直接 `multi_modal2video`
