# script-chunk - 分镜切分技能

> 来源：云盘 jimeng-2026-05-26-6742-script-chunk.md

---

## 技能描述

将完整的拍摄剧本（故事短片剧本/广告营销剧本）内容转化为分镜语言，包含景别、画面描述、声音/台词，确保叙事流畅，画面有张力。

**它是 Phase 5「分镜设计」的第一步**：在完成剧本创作后，将完整剧本按照镜头语言拆分为单个分镜。

---

## 核心原则

1. **完全忠实于剧本**：不增加、不删减剧本原有情节，只做拆分，不做创作
2. **叙事完整性**：按故事发展顺序拆分，保持叙事逻辑连贯
3. **画面可执行**：每个分镜对应一个清晰可拍摄的画面单元，不跨越不同场景
4. **声画对应**：每个分镜必须明确标注对应台词、旁白或音效

---

## 分镜基础要素

每个分镜必须包含以下4要素：

| 要素 | 说明 |
|------|------|
| 镜号 | 从1开始连续编号 |
| 景别 | 标准摄影术语（见下表） |
| 画面 | 描述当前镜头中看到的内容，**必须完全基于剧本描述** |
| 台词/音效 | 对应本镜头的对白、旁白或环境音效 |

### 景别标准

| 景别 | 说明 |
|------|------|
| 远景（EW） | 交代环境，人物占比很小 |
| 全景（FS） | 人物全身，环境完整 |
| 中景（MS） | 人物膝盖以上，兼顾表情和动作 |
| 中近景（MSM） | 人物腰部以上，突出表情 |
| 近景（CU） | 胸部以上，特写表情 |
| 特写（ECU） | 脸部局部/某个部位/某个道具，突出细节 |

---

## 拆分技巧

### 场景切换原则
- 不同场景（时间/地点不同）之间 **必须切分**
- 同一个场景内，根据摄影机位置变化、叙事重点切换进行拆分

### 对话拆分技巧
- 对话双方交替说话时，一般**每句对话对应一个分镜**
- 长对话可以适当合并，但单分镜台词不宜过长
- 如果需要同时表现对话双方的反应，可以切分为对切镜头

### 动作拆分技巧
- 完整连贯的动作可以放在同一个分镜中
- 关键动作转折、重要情绪变化处需要拆分
- 大动作全景 + 小动作特写拆分，强调层次感

---

## 输出格式

```markdown
# {故事名称}分镜清单

| 镜号 | 景别 | 画面描述 | 台词/音效 |
|------|------|----------|-----------|
| 1 | 全景 | 灰蒙蒙的办公室，落地窗被厚重窗帘遮住... | （环境音：时钟滴答声） |
| 2 | 特写 | 桌面上一块金属牌，刻着"遗忘事务所"... | （轻微纸张摩擦声） |
```

---

## 执行检查清单

- ✅ 确认所有分镜按顺序排列，不遗漏任何剧本内容
- ✅ 确认每个分镜景别正确，符合叙事需求
- ✅ 确认画面描述清晰可执行，全部基于剧本原文
- ✅ 确认台词与画面对应，没有放错位置
- ✅ 确认不同场景已经切分开，没有跨场景混编
