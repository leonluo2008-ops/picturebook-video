## 1. Ok 好的 · 小熊领读绘本（2026-06-03 · 飞书云盘素材）

**任务来源**：飞书云盘文件夹 `Wa1vf17nwlGRJSd1aWncJ4m3n1d`（leonluo2008-ops 个人云盘）

**素材**：8 张图（1-8.jpg）+ 1 个"故事简介.txt"（标题 Ok 好的，8 段双语旁白，每句带目标词 OK）

**绘本类型判定**：**半叙事半领读**——
- 8 段、每句带目标词 OK → 领读型特征
- 但有完整叙事弧（起床→出门→吃饭→玩耍→晚安→美好结束）→ 叙事型
- **切分策略**：按 picturebook-video §Phase 5 Step 3 → **2 图合并（v7 范式 v1.2.0+pic21 净化 = 已废）**（弱情节→合并；强情节→1图1Clip）

### 风格锚点（9 维度提炼）

| 维度 | 特征 |
|------|------|
| **媒介技法** | 剪纸拼贴 + 数字绘制 + 轻微水彩晕染（1.jpg 主剪纸；8.jpg 主水彩；混合体）|
| **线条特征** | 圆润、粗黑描边，Q版动物造型 |
| **色彩特征** | 暖色调主导（橙黄棕主色），彩色点缀（红/粉/青蓝/绿/紫），白色描边 |
| **笔触纹理** | 色块拼接 + 颗粒感 + 纸张边缘黑边（剪纸感）|
| **光影处理** | 平面化（剪纸拼贴感强，少真实光影）|
| **构图偏好** | 居中构图，主角在画面中央，文字在主角头顶/脚边 |
| **人物造型** | 拟人化 Q 版动物，幼态（大头、小身、圆眼、腮红），无服装 |
| **整体氛围** | 欢快、温馨、童趣、明亮 |
| **可辨识风格标签** | Eric Carle 拼贴 + Disney 圆头 + 暖色调绘本 |

### 文字呈现（领读模式核心）

- **目标词 OK 作为大型彩色图形元素融入画面**（不是字幕）
- O 字母 = 彩虹色环（红橙黄绿蓝紫扇形）
- K 字母 = 深绿色立体卡纸
- 配中文"好的"在 OK 正下方（深棕色手写体）
- **双行标题布局**：英文 OK 在上，中文"好的"在下
- 位置：画面顶部 1/3

### 通用风格锚点文字（可直接复用）

```
儿童绘本插画风格，剪纸拼贴与水彩晕染结合，圆润 Q 版动物造型，
暖色调（橙黄棕主色），文字作为大型彩色图形元素融入画面（双行标题：
英文在上彩色大字，中文在下深棕手写体），白色描边卡通体，童趣温馨。
```

### 关键观察

- **风格不统一**：1.jpg 偏剪纸拼贴（更具体），8.jpg 偏水彩晕染（更柔和）——绘本作者**混用**两种技法
- **绘本类型影响切分**：领读型用 2 图合并（v7 范式 v1.2.0+pic21 净化 = 已废）后，需要在 prompt 内做"画面切换时机"的设计
- **风格锚点要"压缩"**：一段话能覆盖 9 维度，不能分开写（→ vision 任务会丢维度）
- **v7 范式（已废）适合此绘本**：精准 TTS 卡点（8 段旁白每段明确独立）+ 风格统一（前后页风格锁定）

## 2. Cactus · 仙人掌坚强（2026-06-02 · 仙人掌主题）

**任务来源**：Cactus 4 段分镜 prompt 实测（cactus-clip1-v7.txt + cactus-clips-2-3-4-v7.txt）

**绘本类型**：**领读型**（8 段旁白，每句带目标词 Cactus）—— 仙人掌沙漠绿植主题 + 坚强情感

**BGM 风格**：**A. 静默氛围型**（v7 段 6 写 `No background music, no human voice, no narration, no singing`）—— Cactus 想要沙漠绿植安静氛围

### 风格锚点

| 维度 | 特征 |
|------|------|
| **媒介技法** | 纸艺拼贴（paper-cut craft），可见纸张纹理和毛边 |
| **线条特征** | 圆润粗黑，剪纸剪裁边缘 |
| **色彩特征** | 暖色（沙漠黄、仙人掌绿、暖金）+ 彩色点缀 |
| **笔触纹理** | 色块拼接 + 纸张褶皱 + 颗粒感 |
| **光影处理** | 平面化剪纸，少真实光影 |
| **构图偏好** | 主角居中，背景简洁（沙漠/仙人掌）|
| **人物造型** | 拟人化仙人掌（圆头、Q版、性格化）|
| **整体氛围** | 温暖、童趣、安静、坚强 |
| **可辨识风格标签** | Eric Carle 拼贴（沙漠主题变体）|

### 通用风格锚点文字（已实测 Cactus 4 段全部跑通）

```
Children's picture book collage illustration style, paper-cut craft
with visible paper texture and torn edges, warm and lively atmosphere,
bright primary colors, handcrafted feel.
```

### 关键观察

- **Cactus 是"静默氛围型"代表**——沙漠+绿植+坚强的氛围适合无 BGM
- **段 1 引导句必带**：Cactus 4 段有引导句 → 0 BGM；Ok 好的 4 段缺引导句 → clip2-4 全铺 BGM
- **风格锚点文字越短越稳**——Cactus 5 件套句式实测通过，Ok 好的 5 件套句式跟 Cactus 一致

## 3. Red 苹果 · 认字绘本（2026-06-02 · 苹果认字）

**任务来源**：Red 绘本 v1→v2→v3 三轮修复沉淀（绘本音效-prompt写法.md）

**绘本类型**：**领读型**（目标词 RED + 苹果认字）

**BGM 风格**：**B. 调性匹配型**（用户原话"昨天 Red 绘本效果就非常好"）—— 当时 BGM 是 Seedance 2.0 自带生成的纯音乐

### 风格锚点（参考 picturebook-creator 红苹果案例）

| 维度 | 特征 |
|------|------|
| **媒介技法** | 剪纸拼贴 + 数字绘制 |
| **线条特征** | 圆润粗黑，剪纸剪裁边缘 |
| **色彩特征** | 暖白底色 + 红色主调（苹果红）+ 彩色字母点缀 |
| **笔触纹理** | 色块拼接 + 纸张褶皱 |
| **光影处理** | 平面化，少真实光影 |
| **构图偏好** | 字母作为大型彩色图形元素融入画面（领读核心）|
| **人物造型** | 简单白兔形象 |

## 4. 风格锚点压缩方法论（提炼规则）

> **核心原则**：风格锚点要"压缩"成一段话，能覆盖 9 维度（媒介/线条/色彩/笔触/光影/构图/造型/氛围/标签），不能分开写。

**为什么**：
- vision 任务如果接收"特征列表"会丢维度（模型不知道怎么在画面上同时实现所有特征）
- vision 任务如果接收"长段描述"会丢重点（描述冗长导致模型只能 catch 几个关键词）
- **一段话包含技法层+感觉层**（如"梦幻治愈绘本风格, 水彩晕染与丰富渐变色彩, 物体边缘柔和发光, 色彩化线条勾勒圆润造型"）= 100% 还原作者意图

**错误示范**：
- ❌ 只写技法："paper-cut craft, paper texture, warm color" → 模型画出来可能偏冷（没有感觉层）
- ❌ 只写感觉："warm and dreamy atmosphere" → 模型画出来可能失真（没有技法层）
- ✅ 正确："Children's picture book collage illustration style, paper-cut craft with visible paper texture and torn edges, warm and lively atmosphere, bright primary colors, handcrafted feel"（Cactus 5 件套实测通过）

**自检方法**：写完风格锚点后，**用 9 维度对照检查**——每个维度有覆盖（即使一句话带过 2-3 个维度也行）→ 通过。
