# 案例 #36 · 末尾约束按参考图分 2 类(seedance 文字元素陷阱)

**场景**: Step 5 prompt 末尾约束怎么写。

**翻车**(v5.0.9 → v5.0.11 第 2 次跨本验证):
- 参考图含装饰性文字(字母拼贴/标题/装饰字/角字卡)时,末尾约束写"无字幕、无 Logo" → seedance 照字面执行 "不要文字" = 误删参考图所有文字元素
- "v5.0.9 错误地认为 = 翻译丢字 = 改回中文官方原话就行" = 但 "无字幕" 字面就是 "不要文字"

**修复**(v5.0.11 修订):

**分支 ① 参考图不含文字元素** = 末尾约束 = `无字幕、无 Logo、无水印、无背景音乐`(4 项全写)

**分支 ② 参考图含装饰性文字** = 末尾约束只写 2 项(`无水印、无背景音乐`)+ prompt 每个文字镜头段显式说:
```
"画面 XX 保留参考图中的彩色字母 X 和中文 X 装饰文字作为画面元素自然呈现"
```

**判定口诀**: image-inventory 有装饰性文字 → 走分支 ② → 删无字幕 + prompt 显式说保留

**Step 2 vision 时必查**: 文字元素列(位置 + 颜色 + 类型 + 是否装饰性) → 决定走分支 ① 还是 ②

**对应**: SKILL.md 约束 4 / Step 5 prompt 末尾 / Step 2 vision 自检文字元素列。
