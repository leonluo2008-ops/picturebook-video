# 案例 #26 · 参考图是起点不是限制(v1→v8 反复翻车)

**场景**: Step 5 写 v8 prompt 时,agent 把参考图当"枷锁"严格匹配 5 项(主体位置/朝向/景别/姿态/招牌位置),运镜固定原景别。

**翻车**: 5 项必查"严格匹配" = 僵化 = 模型不知道自由发挥 = 视频单调 = 用户反馈"画面死板"。

**修复**: **参考图是起点不是限制** = 让模型自由推拉切 = 视频流畅。
- ✅ 跑出画面 / 转弯往回跑 / 跑 1-2 秒再换动作 / 姿态一直在变 / 多镜头单镜头多机位
- ❌ "必须保持参考图的整体构图" / "固定原景别不做推拉" / "跑步动作循环"

**判定**: 写 prompt 前 5 项必查 = 启发性参照,**不锁定**。
