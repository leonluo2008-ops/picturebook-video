# 案例 #33 升级版 · 旁白优先逐段合并(权威文档 v1.0)

**场景**: Step 4 Clip 划分。当前 skill 旧做法 = 5 项一致性合并 + 叙事弧决定 clip 数。

**翻车**:
- ❌ 8 图 + 8 段旁白 + 用户 TTS 31s → 按"叙事弧 + 图分组"划分 → 3 图 = 1 clip → 把 31s 均分到 4 clip → clip1 = 6s 装了 3 句旁白 → 6s 读不完 → TTS 画面错位
- ❌ 31s / 4 = 7.75s 不考虑各段旁白长短差异
- ❌ 8 图 = 4 clip(2 图/clip)旁白段长短不一

**修复**(权威文档 5 步):
1. 逐段计算旁白朗读时长(Step 3 输出)
2. 累加相邻段: ≤ 15s → 继续合并;> 15s → 当前组 = 1 clip
3. 向上取整到 [4, 15] 整数
4. 总和 vs 用户 TTS 对齐(差 ≤ 5s ✅)
5. 图跟随旁白段分配

**判定口诀**: "逐段算 → 累加合并 → 取整 → 对齐 TTS → 图跟随"

**优先级**: **最高**(覆盖 #M1 整本节奏 / 5 项一致性合并 / 任何旧版规则)。
