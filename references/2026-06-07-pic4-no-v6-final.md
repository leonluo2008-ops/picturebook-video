## v6 三大核心铁律

### 1. 整数时长铁律（铁律 #72）
- **seedance 不生成小数时长**（用户原话）
- 设计 5.5s → 实跑 6s = 0.5s 浪费
- 设计 5s → 实跑 5s = 0 浪费 ✅

### 2. 彩色文字全程可见铁律（铁律 #73）
- **领读绘本文字 = 领读锚点**，不可一闪而过
- v3-v5 prompt 文字只在前 1-2s 保持
- v6 文字从 t=0 到 t=末帧 5-8s 全程可见 + 微动画

### 3. v5 节奏公式（铁律 #74）
- **总时长 = 朗读完最低 3s + 末帧静默 ≥ 2s**（用户两步推导）
- v6 应用：极短 5s / 短句 6s / 中句 7s / 长句 8s
