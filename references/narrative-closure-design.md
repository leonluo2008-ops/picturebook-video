# 叙事单元收势设计 · 三种模式

## 问题背景
绘本动画场景下，每张静态图是独立叙事单元，图间无物理连续动作。
原问题：clip 视频「突然中断」「突然停车」——动作进行中被截断，没有收势。

根因：video-prompt 的「分镜动作描述」只写了「画面是什么」，没有设计 clip 结尾的「收势动作」。

## 三种收势模式

### 1. 完成式收势
**适用场景**：旁白描述的是一个完整动作/目的

**设计思路**：以「动作完成」收尾，动作本身在本 clip 内完整闭环

**示例**：
- 旁白：「人们用松木和松脂制作火把，保证火焰燃烧得又稳又亮」
- 收势：男子举起做好的火把 → 用火绒点燃 → 火焰稳定燃烧，火光照亮两人脸庞 → **收势在「火焰稳定燃烧」，不是「制作中」被截断**

### 2. 进行式收势
**适用场景**：旁白描述的是持续状态，本 clip 是更大动作的一部分

**设计思路**：Clip 结尾保持运动状态，但给出一个「小目标达成」，为下一个 clip 做铺垫

**示例**：
- 旁白：「夕阳西下，村民们点燃火把，从家门走向田野」
- 收势：队列行走 → 抵达田野边缘 → 小女孩将火炬高高举起 → 众人跟随举起火把 → **收势在「举炬待发」，不是「行进中」被截断**

### 3. 悬念式收势
**适用场景**：需要连接下一个叙事单元，本 clip 是更大叙事的中间环节

**设计思路**：结尾留一个未完成的动作/悬念，但不做大的情绪起伏，保持平稳期待感

**设计原则**：
- 不能用「突然中断」「动作进行一半」这种截断感
- 要给一个「小收束」，比如「停下脚步」「抬头望向远方」「准备做某事」
- 让用户期待下一段，而不是觉得视频坏了

**示例**：
- 旁白：「人们对自然感恩，祈求来年风调雨顺」
- 收势：众人围篝火祈福 → 缓缓放下双手 → 抬头仰望篝火 → **收势在「仰望篝火的宁静神态」，不是「手放下」动作本身被截断**

## 收势设计的时序分布（按情绪类型分档）

> ⚠️ **2026-05-26 更新**：原 15-25% 比例不够，需按情绪浓度分档。
> **核心原则**：生成的 clip 是「素材」，剪辑阶段才出成片。时长设计要为剪辑留冗余，**不能平均划分**。

### 第一档：情感镜头（高情绪浓度）
适用：温暖、传承、感恩、欢庆等情感主题
设计时长：8-10 秒
收势占比：**25-30%**（约 2-2.5 秒）
收势要点：收势阶段要有「余韵感」——镜头缓慢拉远/定格/淡出，给观众情绪沉淀

### 第二档：情绪镜头（中情绪浓度）
适用：有情感但不浓烈、有动作但非激烈
设计时长：6.5-8 秒
收势占比：20-25%（约 1.5-2 秒）

### 第三档：动作/交代镜头（低情绪浓度）
适用：信息交代、快节奏动作、场景转换
设计时长：5-7 秒
收势占比：15-20%（约 1 秒）
要点：快节奏压缩起承，收势快速利落

### 时序分布对照表

| 阶段 | 情感镜头（8-10s） | 情绪镜头（6.5-8s） | 动作镜头（5-7s） |
|------|------------------|-------------------|-----------------|
| 起（开场） | 20-25% | 25-30% | 30-35% |
| 承（发展） | 25-30% | 25-30% | 30-35% |
| 转（转折） | 20-25% | 20-25% | 15-20% |
| 合（收势） | **25-30%** | 20-25% | 15-20% |

### 时长计算示例（火把节实战）

| Clip | 类型 | 旁白字数 | 台词所需 | 情感冗余 | 设计时长 |
|------|------|---------|---------|---------|---------|
| Clip 3（火代表温暖） | 情感 | 18字 | 5.4s | +3.5s | 9秒 |
| Clip 7（篝火欢庆） | 情感 | 20字 | 6.0s | +3s | 9秒 |
| Clip 8（祈福） | 情感 | 22字 | 6.6s | +3s | 10秒 |
| Clip 9（传承） | 情感 | 22字 | 6.6s | +3s | 10秒 |
| Clip 4（火把制作） | 动作 | 18字 | 5.4s | +1s | 6.5秒 |
| Clip 5（队列出发） | 动作 | 16字 | 4.8s | +1s | 6秒 |

**公式**：`设计时长 = max(旁白字数 × 0.3 + 冗余, 4秒)`

### 收势阶段编写规范

**错误**：prompt 只写 `[06-07s] 火焰稳定燃烧`，模型不会自动留余韵。

**正确**：prompt 要明确写出收势的时长和动作质感：
```
[07-09s] 收势阶段（共2秒）：小女孩将火把缓缓举向天空，火焰在夜风中跳跃，镜头缓慢拉远至全景，保持小女孩举火把仰望的静态姿势，停留1.5秒后淡出。
```

三个关键点：
1. **显式标注「收势阶段」** 和时长（如「共2秒」）
2. **写出收尾动作质感**：「缓慢拉远」「定格」「淡出」
3. **写出停留时长**：「停留1.5秒」让模型真正执行静帧

## 错误示例 vs 正确示例

### Clip 4（火把制作）对比

**❌ 错误（无收势）**：
```
@[Image]，场景保持图片一致。
阳光明媚的乡村庭院，一男一女正在制作扫帚/木具，男子用柴刀削木棍，女子整理火把材料。
旁白：人们用松木和松脂制作火把，保证火焰燃烧得又稳又亮。
整体风格保持温暖插画风格。
```
→ 问题：描述的是「制作中」，模型生成出来就是「持续制作→时间到→戛然而止」

**✅ 正确（有收势）**：
```
@[Image]，场景保持图片一致。
[00-02s] 中景，阳光明媚的乡村庭院，男子坐在木凳上用柴刀削松木棍，女子整理火把材料。
[02-04s] 男子将削好的木棍递给女子，女子用松脂绑缚加固，火把雏形完成。
[04-06s] 男子举起完成的火把，用火绒引燃，火焰开始燃烧。
[06-07s] 火焰稳定燃烧，火光照亮两人脸庞，两人对视露出笑容，镜头缓慢拉远。
旁白：人们用松木和松脂制作火把，保证火焰燃烧得又稳又亮。
整体风格保持温暖插画风格，色彩饱和，叙事完整有收尾感。
```
→ 收势在「火焰稳定燃烧，火光照亮两人脸庞」，clip 有头有尾

### Clip 5（夕阳出发）对比

**❌ 错误（无收势）**：
```
@[Image]，场景保持图片一致。
黄昏时分，小女孩举着火炬率村民从家门出发，沿着石板路走向田野。
旁白：夕阳西下，村民们点燃火把，从家门走向田野。
```
→ 问题：只描述了「走向田野」动作刚起就结束

**✅ 正确（有收势）**：
```
@[Image]，场景保持图片一致。
[00-02s] 黄昏石板路，夕阳西下，村民们从家门点燃火把，小女孩高举火炬站在队伍最前方。
[02-04s] 队伍沿着石板路向梯田方向行进，火光在暮色中摇曳跳动。
[04-06s] 队伍抵达田野边缘，小女孩停下脚步，高高举起火炬，回望身后的村民。
[06-07s] 村民们纷纷举起火把，火焰照亮了绿油油的庄稼地，镜头缓缓拉远。
旁白：夕阳西下，村民们点燃火把，从家门走向田野。
整体风格保持温暖插画风格，色彩饱和。
```
→ 收势在「火焰照亮庄稼地」，不是「刚出发」就断

## 核心教训

1. **prompt 里的「分镜动作描述」是核心**：不是写「画面里有什么」，而是设计「这个叙事单元怎么起→怎么承→怎么转→怎么合」
2. **收势动作必须显式描述**：不能靠模型自己推断，要在 prompt 里明确写出 clip 结尾的具体动作状态
3. **收势模式要与旁白叙事功能匹配**：完成式/进行式/悬念式三选一，看旁白说的是「完成某事」还是「持续进行中」还是「中间环节」
