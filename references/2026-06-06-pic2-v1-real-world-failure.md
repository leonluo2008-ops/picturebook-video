## 3 个错位根因（铁律 #49）

### 错位 1 · 末帧"画面继续微动"被 seedance 理解为"画面定格"

**C prompt 文本**：`画面继续微动（小兔子耳朵轻颤 + 妈妈手轻抚书页 + 窗外光影流转）让观众感受呼吸和温馨，末帧 1s 静默消化`

**seedance 实际**：末帧 1s 完全静止，**主角 + 背景全部定格**。

**根因**：seedance 模型把"画面继续微动"理解为**修辞性描写**，不当作**可执行动作指令**。"微动"是抽象词，模型无锚定。

**修复方向（已沉淀到 C 子 agent `_实战校准_末帧微动_具体动作`）**：
- **时间锚定**：`末帧 1s 内：小兔子耳朵轻颤 0.3s + 妈妈手轻抚书页 0.3s + 窗外光影流转 0.4s`
- **可观察动作** ≥1 个

### 错位 2 · 文字"必须完整保留"被 seedance 理解为"软约束"

**C prompt 文本**：`参考图原有的所有文字（顶部彩色英文 "Please" 和中文"请"字）必须完整保留作为画面元素，模型不得删除或替换这些文字，让文字自然融入场景`

**seedance 实际**：frame_03 顶部 Please + 请字 **完全消失**。

**根因**：seedance 模型把"必须完整保留"理解为**修辞强调**（"让 X 自然融入场景"暗示"可以自然消失"），不当作**空间位置锁定**。文字没有"位置"约束，模型可以自由重绘顶部 1/6 画面。

**修复方向（已沉淀到 C 子 agent `_实战校准_文字保留_锁定位置`）**：
- **空间位置锁定**：`参考图原有的所有文字（顶部 1/6 画面的彩色拼贴英文 "Please" 和中文"请"字）必须完整保留，文字位置和颜色不得改动`
- 备选：`--image` 首帧模式钉死构图（铁律 #40 方案 A）

### 错位 3 · 镜头一"切到全景"被 seedance 理解为"远景剪掉主体"

**C prompt 文本**：`镜头一（0-1s · 场景建立）：镜头切到室内阅读空间全景，纸艺拼贴纹理清晰可见`

**seedance 实际**：frame_01 **主体完全丢失**（画面只剩米色背景 + 窗帘角）。

**根因**：seedance 模型把"切到全景"理解为"拉远 + 剪掉主体"。"全景"被解读为"wide shot"（远景），不是"full shot"（全身）。

**修复方向（已沉淀到 C 子 agent `_实战校准_镜头一_主体完整可见`）**：
- **显式主体约束**：`镜头一（0-1s · 场景建立）：镜头切到全景室内阅读空间，两只兔子（橙兔+棕兔）必须完整可见于画面中心，纸艺拼贴纹理清晰可见`
- 备选：`--image` 首帧模式钉死构图

## 实战翻车反思

**1. evals 评分方法本身的盲区**：
- evals 测的 = C 输出 JSON **文本合规性**（"末帧包含'微动'字样" → ✓）
- 实战测的 = seedance **实际行为**（"末帧是不是真的在动" → ❌）
- **evals 9 维度全胜 ≠ v1.0.0 可用**——这是 evals 评分方法的盲区，**不是 v1.0.0 重构本身的错**

**2. 修复方向 = C 升级 self_check + D 升级自检**：
- C 自我升级 self_check 9 → 12 项（+3 个 _实战校准 实战错位检查）
- D 对齐 12 项自检（实战校准 = 必查项）
- 未来 v1.0.1 重跑时，C 12/12 + D 6/6 都过 = 真可用

**3. 重跑计划**：
- 修 C 子 agent 3 个措辞后，重跑 Pic2 Clip 1
- 若 Clip 1 实战 self_check ≥5/6 → 批量跑 Clip 2-7 + Clip 8
- 若 Clip 1 仍翻车 → 改用 `--image` 首帧模式（铁律 #40 方案 A）
