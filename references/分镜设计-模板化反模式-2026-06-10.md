## 2. 3 个 Step 决定规则（v1.2.0+pic22 新增 · 必走）

### Step A · 看图决定景别 + 运镜

**核心问题**：镜头 1 应该写什么运镜 = 取决于**画面实际状态**，**不**取决于"建立场景"安全帽。

| 画面状态 | 镜头 1 推荐运镜 | 反例（❌ 模板化）|
|---|---|---|
| **主角在画面中央静态**（仓鼠蹲伏、仓鼠站桩）| **固定中景** + 不动 = "仓鼠就是舞台中心" | ❌ "中景拉远定格"（仓鼠本来就在中央 = 拉什么？）|
| **主角从洞口/草丛探出**（木屋门洞、草丛藏身）| **侧面机位看洞口**（找一找）| ❌ "中景拉远"（洞口才是视觉焦点）|
| **主角在跑轮/木屋里动**（跑轮、跑动、玩耍）| **正面平视 + 不推不拉**（让跑轮/木屋本身作动感）| ❌ "中景拉远"（跑轮已经填满画面）|
| **主角在开阔户外**（草地、空地、花田）| **中景拉远**（交代户外空间感）| ❌ 拉远 = OK · **但不是默认** |
| **主角在角落小场景**（钻木屑、藏角落）| **近景特写**（捕捉细节动作）| ❌ 拉远（看不到细节）|

**判断口诀**：**"看图先决定景别"**（Step A）

### Step B · 看图决定音效

**核心问题**：音效**不**是"<一声轻柔的微风声>"作为默认兜底 = 取决于**场景真实环境声音**。

| 画面场景 | 推荐音效 | 反例（❌ 模板化）|
|---|---|---|
| **户外草地 / 花田** | <微风声> / <鸟叫> / <远处铃声> | ✅ OK（自然户外）|
| **木屑堆 / 笼舍里** | <细碎刨花声> / <仓鼠呼吸声> / 静默 | ❌ "微风声"（笼舍里没风）|
| **跑轮 / 跑动中** | <跑轮踏板吱呀声> / <脚步轻踏声> | ❌ "微风声"（跑动是动态声）|
| **木屋门洞 / 室内** | <木屋吱呀声> / <仓鼠的呼气声> / 静默 | ❌ "微风声"（室内没风）|
| **静态画面**（仓鼠呆萌蹲着）| **静默**（让画面安静）| ❌ 任何声音 = 喧宾夺主 |

**判断口诀**：**"看图先决定音效"**（Step B）

### Step C · 镜头命名用"动作+状态"

**核心问题**：镜头 1 **不**叫"建立场景" = 叫"动作+状态"（蹲伏姿态 / 侧面看门洞 / 跑轮开跑 等）。

- ❌ "镜头 1（建立场景·0-2s）"（每个镜头 1 都长这样 = 模板化）
- ✅ "镜头 1（蹲伏姿态·0-2.5s）" / "镜头 1（侧面看门洞·0-2s）" / "镜头 1（跑轮开跑·0-2s）"

**判断口诀**：**"看图先决定镜头名"**（Step C）

## 3. 6 段 prompt 重写示例（按 3 个 Step 修复）

### 修复版 clip3 镜头 1（仓鼠蹲伏静态）
```
镜头 1（蹲伏姿态·0-2.5s）：固定中景，画面不再动。<主角小仓鼠>正安静地蹲伏在浅米黄色木屑堆上，四肢着地，圆润大眼望向镜头，粉色小鼻微微翕动，周围是微观小世界——左侧木屋顶尖一角露出，右侧银色跑轮静立，<一声细碎刨花声>。
```
✅ 看图：仓鼠蹲伏 = 固定中景（Step A）· 笼舍里 = 刨花声（Step B）· 镜头名"蹲伏姿态"（Step C）

### 修复版 clip5 镜头 1（草地啃瓜子）
```
镜头 1（草地上开吃·0-2s）：正面中景平视。<主角小仓鼠>坐在明亮草地上，双手捧着一颗葵花籽举到嘴边准备啃食，眼睛眯成月牙形笑，腮红粉嫩，左下角浅棕色木碟里散放着 3 颗葵花籽，<一声轻柔的微风声>。
```
✅ 看图：草地 + 吃 = 正面平视（Step A）· 户外草地 = 微风声 ✅（Step B）· 镜头名"草地上开吃"（Step C）

### 修复版 clip6 镜头 1（跑轮里跑）
```
镜头 1（跑轮开跑·0-2s）：正面平视银色金属跑轮。<主角小仓鼠>在跑轮内侧跑道上，身体压低，四肢张开踏在跑道踏板上，头朝左前方，耳朵微微后倾，呈奔跑中的动势，右侧木屋顶尖和饮水瓶静静立着，<跑轮踏板轻微的吱呀声>。
```
✅ 看图：跑轮居中 = 正面平视不推不拉（Step A）· 跑动 = 跑轮踏板声 ❌不写微风（Step B）· 镜头名"跑轮开跑"（Step C）

### 修复版 clip7 镜头 1（木屋门洞探出）
```
镜头 1（侧面看门洞·0-3s）：侧面机位看木屋门洞。<主角小仓鼠>从门洞里探出大半个身子，四只粉色小爪都落在木屑上，前爪微微撑起身体，耳朵竖立呈好奇状，目光朝向镜头外看，胡须向两侧舒展，<一声木屋吱呀声>。
```
✅ 看图：洞口 = 侧面机位（Step A）· 室内 = 木屋吱呀声 ❌不写微风（Step B）· 镜头名"侧面看门洞"（Step C）

## 5. 完整 prompt 写作流程（v1.2.0+pic22 升级版）

```
Step 0 · 接收需求（绘本 + 图 + 旁白 + 平台）
   ↓
Step 1 · vision_analyze 每张图（**强制必看**）
   ↓  ← 这一步之前**没强制** = 模板化根因
Step 2 · 按 4 步原则写 prompt
   ↓
   Step 2.1 · Step A 看图决定景别 + 运镜（5 种画面状态 → 5 种运镜）
   Step 2.2 · Step B 看图决定音效（5 种场景 → 5 种音效，**禁微风为默认兜底**）
   Step 2.3 · Step C 镜头名用"动作+状态"（禁"建立场景"标签）
   Step 2.4 · 写动作步骤数（1-3 镜/段自由组合 · **不**套 3 镜模板）
   Step 2.5 · 末帧写"画面不再动 · 让 XX 在小朋友心中沉淀"（**真正静默不是微动**）
   Step 2.6 · 末尾约束段用官方原话 4 词（无人声/无歌唱/无配音/无朗读）
   Step 2.7 · 无 TTS 元数据术语（禁 TTS 音轨占位/音轨占位/朗读时长）
   ↓
Step 3 · 单 Clip 端到端先验（不批量跳跑）
   ↓
Step 4 · 3 并发跑（picturebook-workflow.md 铁律）
   ↓
Step 5 · 发飞书 + 完整证据链
```
