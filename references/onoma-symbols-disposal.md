# 拟声符号（声波/动作线/爆炸星）= 显式说"出现后立即消失"

> **铁律**：漫画/绘本参考图中的"拟声符号"（sound waves / action lines / 爆炸星）= 视觉化的"嘎嘎叫/砰"——在视频中**叫完就要消失**，不是持续物体。

---

## 1. 拟声符号的识别

**常见类型**：
- 声波（sound waves）= 头部/嘴部周围的彩色弧线或竖条（如参考图头部的橙黄绿 3 道彩条）
- 动作线（action lines）= 表示速度/冲击的直线
- 爆炸星（impact stars）= 表示碰撞/惊讶的星形
- 汗滴/青筋（sweat drops / vein marks）= 表示紧张/无奈
- 拟声词文字（GABBA / POW / BOOM）= 字体艺术化的音效文字

**判定口诀**：
- **持续物体**（如鸭子身体、远处小木牌、背景草地）= 全程保留
- **拟声符号**（彩条、动作线、爆炸星）= 出现即逝，叫完/动完/撞完 = 消失

---

## 2. 真实翻车案例（2026-06-14 绘本中段 clip）

**参考图**：黄色鸭子立草地，头部有 **3 道橙黄绿竖条**（声波），表示"嘎嘎叫"。

**原 prompt 写法（错）**：
```
镜头 1：参考 @Image4，一只黄色鸭子站在草地上，镜头从远景缓慢推近到中景，
        鸭子张开嘴巴发出嘎嘎叫声，叫声后自然合上嘴巴。
镜头 2：参考 @Image4 的同一只黄色鸭子，镜头切到中景，鸭子的头轻微上下点动一下，
        嘴巴再次张开嘎嘎叫，然后合上。
```
> **完全没提那 3 道彩条**。

**翻车结果**（抽帧验证 t0/t2/t4/t5/t6）：
- t0 帧：3 道彩条还在
- t2 帧：3 道彩条还在
- t4 帧：3 道彩条还在
- t5 帧：3 道彩条还在
- t6 帧：3 道彩条还在
- **从头到尾 8 秒一直挂着** = 翻车

**根因**：模型没收到"3 道彩条是拟声符号，叫完要消失"的指令 → 把它当成"持续发声动作的一部分" = 全程维持。

## 3. 正确写法（重做后修复）

**v7 prompt 关键句**：
```
镜头：参考 @Image4，固定中景（不推不切不拉）。鸭子张开橙黄色嘴巴嘎嘎叫一声，
     头部 3 道橙黄绿彩色声波符号（漫画拟声）短暂出现 0.5 秒后立即消失，
     鸭子合上嘴巴，画面回到平静的草地站立状态。
```

**关键点拆解**：
1. **显式提到元素**："3 道橙黄绿彩色声波符号" = 模型知道是哪个元素
2. **标注语义**："（漫画拟声）" = 告诉模型"这是叙事符号不是物体"
3. **规定时长**："短暂出现 0.5 秒" = 限定出现窗口
4. **规定动作**："后立即消失" = 明确消失指令

---

## 4. v7 抽帧验证（实测成功）

| 帧 | 声波 | 嘴巴 | 解读 |
|---|---|---|---|
| t0 | ❌ 消失 | 张开 | 即将叫 |
| **t1** | **✅ 出现**（橙黄绿 3 道）| 张大 | **叫 + 声波** |
| **t2** | **✅ 出现**（同 t1）| 张大 | 声波持续 |
| **t3** | **❌ 消失** | **合上** | **叫完收声** |
| t4 | ❌ 消失 | 合上 | 末帧静止 |

**结果**：声波 t1-t2 出现，t3 消失 = **完全符合 prompt 设计**。

## 5. image-inventory 必含标注

**在 `image-inventory.md` 写"有/没有"清单时，**额外加一列标注"持续物体 vs 叙事符号"**：

```markdown
# 图 | ... | **有** | 元素类型（持续物体/叙事符号）
4 | 纸艺拼贴 | 1 大鸭·嘴巴大张·**头部 3 道彩条** | **叙事符号（声波·出现即逝）** / 鸭子身体 / 草地 / 远处小木牌（持续物体）
```

**判定标准**：
- 元素 = **物理实体**（身体/草/木牌/水）→ 持续物体
- 元素 = **视觉化音效/动作/情绪符号**（彩条/动作线/汗滴/星形/拟声字）→ 叙事符号

---

## 6. Prompt 通用模板

**所有含"叙事符号"的镜头 = 必含 3 要素**：

```
1. 显式提到元素（具体描述）："头部 3 道橙黄绿彩色声波符号"
2. 标注语义："（漫画拟声）" / "（视觉化动作线）" / "（碰撞爆炸星）"
3. 规定动作："短暂出现 0.5 秒后立即消失" / "出现 1 秒后随动作完成而消失"
```

**反例**（禁止）：
```
❌ 完全不提（= 模型不知道怎么处理 = 持续挂着）
❌ 只说"不要有彩条"（= 约束词污染 = 模型过度反应 = 永远不出现）
```

**正例**：
```
✅ "头部 3 道橙黄绿彩色声波符号（漫画拟声）短暂出现 0.5 秒后立即消失"
```
