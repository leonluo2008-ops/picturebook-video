# v8.1 提示词结构蓝本（Potato 优 · v5.0.11）

> **来源**: Potato 2026-06-24 实测效果优，Lettuce 2026-06-24 实测效果劣
> **核心结论**: Potato 提示词结构 = 优蓝本。4 个核心差异对应 4 条强制规则 R6/R7/R8/R9。
> **用途**: 新 prompt 创作时直接参照此蓝本的段落结构 copy。

---

## 1. 5 条强制段约束（R6/R7/R8/R9/R10）

| 规则 | 段 | 强制要求 | 为什么 |
|------|----|----------|--------|
| **R6** | 段 1 · 视觉基底 | 每张参考图**分别描述**视觉基底（位置/景别/视觉特征/文字元素），多图之间用"**接着**"衔接 | seedance 需要逐图差异才能做镜头切换，只写 `@Image1 @Image2` 不分别描述差异 = 画面模糊 |
| **R7** | 段 2 · 动作序列 | 每个镜头必须用 `镜头 N（@ImageN + 段 N 旁白）：` 格式**显式绑定**参考图和旁白段 | seedance 需要知道每个镜头对应哪张参考图、对应哪段旁白，不绑定 = 画面动作和旁白不同步 |
| **R8** | 段 3 · 旁白-动作对应 | 必须用 `段 N 念"完整旁白文本"：动作描述` 格式，**逐段列出完整旁白原文** + 动作映射 | seedance 需要旁白原文才能直接对照，只写"念到 X 词时"太模糊 = 对不上 |
| **R9** | 段 4 · 收尾 | 必须用 `镜头 N 末尾：` 指明位置 + **具体微动描述**，不可用"回到开场状态"等抽象语 | seedance 需要精确知道收尾在哪个镜头之后、做什么动作，抽象语 = 画面趋向凝固 |
| **R10** | 段 5 · 音频描述(generate_audio=True 时,v5.0.13 路由) | 必须新增 **段 5 · 音频描述**。路径 A(有 SRT): 写 `人声旁白："{SRT原文}"` + `音效：{具体音效描述}`。路径 B(无 SRT): 只写 `音效：{具体音效描述}`(人声旁白可选,seedance 不知道说什么)。路径 C/D(spike/静默): 整段省略。**永远不生成 BGM** | seedance generate_audio=True 时会自动生成音频,不写音频描述段 → seedance 自己猜内容和节奏 → 不可控/与 SRT 不匹配 |

## 2. Potato 优蓝本（4 个 Clip 完整模板）

### Clip 1 模板

```
【参考图 @Image1 + @Image2】

【段 1 · 主体 + 视觉基底】
一颗橙棕色椭圆土豆（@Image1 正面特写），带笑脸和黑色斑点。画面上方有彩色"potato"字母
（红/黄/绿/蓝渐变），下方有"土豆"中文（绿/黄/红色装饰字体），土黄背景。

接着同一颗土豆变成深棕色（@Image2 正面特写），上方彩色"potato"字母，下方"土豆"中文
（黑色字体），黄橙背景。

【段 2 · 主体动作序列】
镜头 1（土豆 @Image1 + 段 1 旁白）：
土豆正面居中，脸上的笑脸弯弧度缓慢加深，黑色斑点轻微抖动。"potato"字母随朗读顺序逐个
轻轻弹跳，"土豆"二字同步轻摆。念完段 1 后画面短暂呼吸。

镜头 2（土豆 @Image2 + 段 2 旁白）：
土豆身形缓慢变深（橙棕 → 深棕），表面纹理逐渐显现，斑点变明显。念到"棕色"时颜色加深
最明显。念完段 2 后土豆轻摆一次回到中央，黑点继续轻微呼吸。

【段 3 · 旁白-动作对应】
段 1 念"土豆potato"：土豆正面展示 + 字母轻弹对应"potato"
段 2 念"一颗棕色土豆potato"：颜色加深对应"棕色" + 字母轻弹对应"potato"
两段之间是 1 秒多的呼吸停顿，土豆静止画面。

【段 4 · 收尾不凝固】
镜头 2 末尾：土豆轻摆后回到中央，黑色斑点继续微微抖动，"potato"字母尾字母 o 轻轻一弹后
回弹一次，保持微动状态循环过渡到下一段。

【段 5 · 音频描述】（generate_audio=True 时必写）
人声旁白："土豆potato" — 温柔女声，语速偏慢，领读节奏清晰。
音效：字母轻弹时配轻微"叮"声，土豆斑点抖动时无音效。
人声旁白："一颗棕色土豆potato" — 同上温柔女声，"棕色"处略微加重语气。
音效：颜色加深时配低沉"嗡"声。

无水印、无背景音乐。
```

## 3. Lettuce 劣反例对照（4 个核心问题）

### 3.1 段 1 · Lettuce 只写主体没逐图描述

**Lettuce（❌）**:
```
一颗绿色生菜，叶片层层叠叠，圆润饱满，特写（参考图 @Image1 @Image2），画面上方居中保留
彩色"lettuce"和"生菜"装饰文字。
```

**问题**: @Image1 和 @Image2 并列写出，但没有分别描述两张图的视觉差异（背景色/文字样式/主体姿态）。seedance 不知道两张图有什么不同，无法做镜头切换。

**Potato 对标（✅）**: 逐图分别描述——图 1 土黄背景+绿黄红文字，"接着"图 2 黄橙背景+黑色文字。

### 3.2 段 2 · Lettuce 镜头没绑定旁白段

**Lettuce（❌）**:
```
镜头 1（开场）：生菜居中，圆润叶片微微舒展...
镜头 2：一只小手从画面右侧伸入...
```

**问题**: 镜头编号用"开场"而非段号，没绑定 @ImageN 和"段 N 旁白"。seedance 不知道镜头 1 对应哪段旁白文本。

**Potato 对标（✅）**: `镜头 1（土豆 @Image1 + 段 1 旁白）：` — 图+段+旁白三绑定。

### 3.3 段 3 · Lettuce 没列完整旁白文本

**Lettuce（❌）**:
```
念到"生菜"时生菜居中完整呈现
念到"圆圆的"时镜头略微推近展示生菜圆润轮廓
念到"Lettuce"时画面慢速拉远展示整体
```

**问题**: 只说"念到 X 词时 Y 动作"，没把段 N 的完整旁白文本列出来。seedance 需要旁白原文才能直接对照时间点。

**Potato 对标（✅）**: `段 1 念"土豆potato"：土豆正面展示 + 字母轻弹对应"potato"` — 完整旁白 + 动作映射。

### 3.4 段 4 · Lettuce 收尾用抽象语

**Lettuce（❌）**:
```
镜头 4（收尾）：回到开场状态，生菜叶片继续微风轻摆，光斑缓慢移动，小水珠停留在叶尖微微晃动。
```

**问题**: "回到开场状态"是抽象的，没说明在哪个镜头后收尾。虽然后面有微动描述，但位置不明确。

**Potato 对标（✅）**: `镜头 2 末尾：土豆轻摆后回到中央，黑色斑点继续微微抖动...` — 具体镜头位置 + 具体微动。

## 4. 结构化段落骨架（通用模板）

> 以下是不含具体内容的骨架，新 prompt 创作时直接填空。

```
【参考图 @ImageN + @ImageN】

【段 1 · 主体 + 视觉基底】
{主体描述}（@Image1 {景别}），{视觉特征}。{文字元素描述}，{背景描述}。

接着{同一/不同主体变成/切换到}（@Image2 {景别}），{视觉特征差异}。{文字元素差异}，{背景差异}。

【段 2 · 主体动作序列】
镜头 1（{主体} @Image1 + 段 {N} 旁白）：
{主体具体动作}，{视觉细节变化}。念完段 {N} 后{过渡描述}。

镜头 2（{主体} @Image2 + 段 {N} 旁白）：
{主体具体动作}，{视觉细节变化}。念到"{关键词}"时{视觉呼应}。

【段 3 · 旁白-动作对应】
段 {N} 念"{完整旁白文本}"：{动作映射}
段 {N} 念"{完整旁白文本}"：{动作映射}
两段之间约 {X} 秒{停顿描述}。

【段 4 · 收尾不凝固】
镜头 {N} 末尾：{具体微动描述}，保持{运动状态}过渡到下一段。

【段 5 · 音频描述】（generate_audio=True 时必写，领读型/generate_audio=False 时省略）
人声旁白："{完整旁白文本}" — {声线描述，语速，节奏}。
音效：{与画面动作对应的音效描述}。

{末尾约束分流}
```

## 5. verify_prompt.py 规则映射

| 规则 ID | 检查函数 | 严重级 | 检测逻辑 |
|---------|----------|--------|----------|
| R6 | `check_per_image_description()` | FAIL | 段 1 必含 N 处 `@ImageN` 各自独立的描述句 + "接着"衔接 |
| R7 | `check_shot_narration_binding()` | FAIL | 段 2 每个镜头标记必含 `@ImageN` + `段 N 旁白` 绑定 |
