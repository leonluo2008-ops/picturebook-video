# 绘本 v14 范式 · 4 段默认 prompt 结构（2026-06-04 起绘本默认）

> **v14 是绘本启动的默认范式**（不是 v7/v8/v9/v10/v11）。
> 适用场景：**任何绘本**（领读型、叙事型、单词认知型、情绪绘本等）—— 静态图 → Seedance 视频的通用 4 段 prompt 结构。
> v7/v8/v9/v10/v11 是**音频策略的子范式**（v14 内部嵌入）。

## v14 核心定位

| 维度 | 范式 |
|---|---|
| **绘本通用骨架** | **v14**（主体定义 + 分镜绑定 + 风格 + BGM 段） |
| **音频策略（嵌入 v14 段 4）** | v7 静默型 / v8 调性匹配型 / v9 整 Clip 一致型 / v10 领读型跨 Clip 同主题 |
| **收势** | v14 段 2 必含 `final frame holds to the last frame` |
| **文字保留** | v14 段 2 必含 v2 措辞（参考图所有文字作为画面元素自然融入） |
| **风格** | v14 段 3 必含 `2D paper collage style`（拼贴风统一锚点） |

## v14 标准 4 段结构

### 段 1 · 主体定义（Character + Style Anchors）

**作用**：把参考图里**所有视觉元素**（角色 + 场景 + 装饰 + 文字）描述清楚，作为 Seedance 视觉理解的"画布"。

**写法**：
- 第一句：角色描述（外观 + 颜色 + 质感）
- 中间：场景背景（天空 / 地面 / 装饰元素）
- 文字：参考图里的所有文字（含颜色 + 位置 + 风格）
- 收尾：风格锚点词（手绘/拼贴/水彩/写实）

**示例**（Say说 封面 P1）：
```
一只可爱的卡通小熊，由深浅棕色卡纸拼贴而成，圆头圆耳，黑色圆眼睛，浅色口鼻部，双爪举到脸颊两侧作呼喊姿势，嘴巴大张呈 O 形；画面上方居中是彩色拼贴字母 Say（S 红、a 黄、y 绿），下方紧贴中文"说"字（金黄色），四周散落彩色音符和白色对话气泡；背景为浅蓝渐变天空 + 鲜绿草地，整体是儿童绘本封面节奏。
```

### 段 2 · 分镜绑定（Movement + 收势 + 文字保留 v2）

**作用**：规定**这一段视频里发生什么动作** + 必含 v2 文字保留措辞 + 必含收势。

**写法**（4 个子要素，用分号 `;` 串接）：
1. **`@图N 作为唯一参考帧`**（v14 关键：单图连续运镜，不切换其他图）
2. **镜头运镜**（中景→中近景 / 缓慢推进 / 拉远 / 微平移）
3. **v2 文字保留措辞**（必须原样）：
   ```
   参考图原有的所有文字（包括标题/领读文字/画面内的任何中英文）作为画面元素自然融入场景，模型自行设计合理的呈现节奏与动效（例如淡入、随镜头呼吸、配合动作出现等），不重新生成任何新文字。
   ```
4. **起承转合 + 收势**（4 段叙事弧 + 收势词在最后一句画面描述）
   ```
   起：[起始状态描述 1.5s]；
   承：[中段动作 3s]；
   转：[高潮变化 2-3s]；
   合：画面最终定格在 [收势状态]，镜头完全静止不动，不渐隐不淡出，停留至最后一帧；
   ```
5. **音效副词**（嵌入视觉句中间，逗号 `,` 串接）：
   ```
   起：小熊双手捂嘴定格 1.5 秒，营造封面"呼喊"瞬间；
   承：镜头缓推 3 秒，露出更多草地和音符细节，伴随轻柔的纸张摩擦"沙沙"声；
   转：小熊表情强化，眼睛亮起，嘴角微扬，伴随一声清脆的"叮"；
   合：画面最终定格在小熊双手捂嘴+Say 字母饱满呈现的姿态，镜头完全静止不动，不渐隐不淡出，停留至最后一帧；
   ```

### 段 3 · 风格锁定（Style Anchor）

**作用**：统一整段视频的视觉风格，避免 Seedance 跑偏。

**标准锚点**（纸艺拼贴风绘本默认）：
```
2D paper collage style, 儿童绘本纸艺拼贴风，剪纸 + 撕纸 + 卡纸拼贴质感，边缘可见纸张纤维，色彩饱和明亮，童趣活泼。
```

**变体**（按绘本实际风格调）：
- 写实风：`Photorealistic watercolor style, 柔和自然光，细腻质感`
- 数字插画：`Flat digital illustration, bold colors, clean lines`
- 水彩手绘：`Soft watercolor style, gentle brushstrokes, dreamy atmosphere`

### 段 4 · BGM 段（音频策略子范式选择）

**作用**：决定这段视频的音频层（旁白 / BGM / 音效 / 人声禁令）。

**v14 BGM 二选一**（2026-06-04 起标准）：

| 模式 | 适用 | 写法 |
|---|---|---|
| **A. 传 mp3** | 用户有 BGM 文件 | `--audio ./bgm.mp3` + prompt BGM 段写"持续铺底轻柔 BGM" |
| **B. 不传 mp3**（推荐） | 用户没有 BGM / 想要 Seedance 自带生成 | 不传 `--audio` + 删 BGM 描述词 + 改写为"仅环境音效无 BGM"或"让 Seedance 配相符调性纯音乐" |

**音效永远有**（`--generate-audio true`），BGM 看用户。

**音频策略子范式选择**（v7/v8/v9/v10）：

| 子范式 | 适用 | 段 4 写法 | 实测案例 |
|---|---|---|---|
| **v7 静默型** | 安静 / 治愈 / 单色系 | `No background music, no human voice, no narration, no singing;` | Cactus（沙漠仙人掌） |
| **v8 调性匹配型** | 多情绪 / 暖色 / 复杂 | 删 `No background music`，让 Seedance 配调性相符纯音乐 | Red 苹果、Ok 好的 |
| **v9 整 Clip 一致型** | 同主题多动作 | 整 Clip 一段 BGM + `continues throughout the entire clip` + `mood shift` | Eat 吃 |
| **v10 领读型跨 Clip** | 重复句式绘本 | 整 Clip 一段 BGM + clip 2+ 写 `same warm ... from the previous clip continues` | Eat 吃 clip 2/3 |

## v14 完整示例（Say说 封面 P1 · v7 静默型）

```
[段 1 主体定义]
一只可爱的卡通小熊，由深浅棕色卡纸拼贴而成，圆头圆耳，黑色圆眼睛，浅色口鼻部，双爪举到脸颊两侧作呼喊姿势，嘴巴大张呈 O 形；画面上方居中是彩色拼贴字母 Say（S 红、a 黄、y 绿），下方紧贴中文"说"字（金黄色），四周散落彩色音符和白色对话气泡；背景为浅蓝渐变天空 + 鲜绿草地，整体是儿童绘本封面节奏。

[段 2 分镜绑定]
@图1 作为唯一参考帧，镜头从中景缓慢推进到中近景聚焦小熊脸部，字母 Say 在画面上方稳定存在，模型自行设计合理的呈现节奏与动效（例如轻微呼吸、随镜头放大等），参考图原有的所有文字（包括彩色字母 Say、中文"说"、所有装饰文字）作为画面元素自然融入场景，不重新生成任何新文字；起：小熊双手捂嘴定格 1.5 秒，营造封面"呼喊"瞬间；承：镜头缓推 3 秒，露出更多草地和音符细节，伴随轻柔的纸张摩擦"沙沙"声；转：小熊表情强化，眼睛亮起，嘴角微扬，伴随一声清脆的"叮"；合：画面最终定格在小熊双手捂嘴+Say 字母饱满呈现的姿态，镜头完全静止不动，不渐隐不淡出，停留至最后一帧；

[段 3 风格锁定]
2D paper collage style, 儿童绘本纸艺拼贴风，剪纸 + 撕纸 + 卡纸拼贴质感，边缘可见纸张纤维，色彩饱和明亮，童趣活泼。

[段 4 BGM 段 · v7 静默型]
No background music, no human voice, no narration, no singing, no humming, no whistling, only natural sound effects (chimes, paper rustling, gentle taps).
```

## v14 必跑 11 项自检

- [ ] **段 1 必含角色 + 文字 + 场景 + 风格锚点**（4 要素齐全）
- [ ] **段 2 必含 `@图N 作为唯一参考帧`**（v14 单图连续运镜核心）
- [ ] **段 2 必含 v2 文字保留措辞**（v2 完整三要素）
- [ ] **段 2 收势词在最后一句画面描述**（`final frame ... holds to the last frame`）
- [ ] **段 2 收势词后不追加任何画面动作**（收势是绝对末尾）
- [ ] **段 2 整段用分号 `;` 串接**（v14 = 3 个分号：段 1/2/3/4 之间）
- [ ] **段 2 视觉句中无 `[Sound effect: ...]` 独立块**（音效嵌入逗号）
- [ ] **段 2 视觉句中无否定句**（`no speech` / `no narration` / `no subtitles`——会消参考图文字）
- [ ] **段 3 必含 `2D paper collage style` 或对应风格锚点**
- [ ] **段 4 BGM 子范式选择正确**（v7 静默 / v8 调性 / v9 整 Clip / v10 跨 Clip）
- [ ] **参数必加 `--watermark false` + `--generate-audio true`**

## v14 决策树（绘本启动必跑）

```
绘本类型？
├── 领读型（弱情节 + 重复句式，如 Say说 / I eat X / Pete the cat）
│   ├── 整体氛围一致优先 → **v14 + v10 跨 Clip 同主题**
│   └── 单 Clip 内多情绪 → **v14 + v8 调性匹配型**
├── 单词认知型（Look at the X / 颜色认知）
│   ├── 静默治愈 → **v14 + v7 静默型**
│   └── 活泼有 BGM → **v14 + v9 整 Clip 一致型**
├── 叙事型（强情节 + 起承转合）
│   └── **v14 + v7 静默型**（默认无 BGM，叙事靠画面推）
└── 情绪绘本（睡前 / 治愈 / 暖心）
    └── **v14 + v7 静默型**（氛围优先，不加 BGM）
```

## v14 与 v7/v8/v9/v10 的关系

| 范式 | 定位 | 关系 |
|---|---|---|
| **v14** | 绘本默认骨架（4 段 prompt） | 母范式 |
| **v7** | 静默型音频子范式 | v14 段 4 用 v7 写法 |
| **v8** | 调性匹配型音频子范式 | v14 段 4 用 v8 写法（删 No background music） |
| **v9** | 整 Clip 一致型音频子范式 | v14 段 4 用 v9 写法（整 Clip 一段 BGM + mood shift） |
| **v10** | 领读型跨 Clip 同主题 | v14 段 4 用 v10 写法（clip 2+ 写 same ... from previous clip） |

**核心**：v14 是骨架，v7-v10 是 v14 段 4 的音频子策略。绘本启动默认走 v14 骨架 + 选一个音频子范式。

## v14 必避反模式

- ❌ **v14 段 2 不用 `@图N`**（变成无参考的纯文本生成，风格跑偏）
- ❌ **v14 段 2 漏 v2 文字保留措辞**（参考图文字被消）
- ❌ **v14 段 2 收势词后追加音效块**（覆盖收势指令，画面不静止）
- ❌ **v14 段 2 用 `[Sound effect: ...]` 独立块**（句号把视觉切成多段，节奏拖乱）
- ❌ **v14 段 3 漏 `2D paper collage style` 锚点**（风格跑偏成写实/3D）
- ❌ **v14 段 4 BGM 段和视觉段顺序反**（BGM 必须在风格段之后）
- ❌ **v14 缺 `--watermark false`**（绘本带 AI 水印=产品缺陷）
- ❌ **v14 缺 `--generate-audio true`**（绘本 ≠ 全静音，音效必须生成）
