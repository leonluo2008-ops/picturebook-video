## 1. 触发条件（什么时候需要拆）

| 情况 | 单图朗读 | + 必要留白 | 总时长 | 处理 |
|---|---|---|---|---|
| 不需要拆 | < 5s | + 5s | < 10s | 1 Clip 搞定 |
| 临界 | 5-8s | + 7-9s | 12-17s | 看是否 ≤ 15s |
| **必须拆** | **> 8s** | **+ 10s+** | **> 15s** | **多 Clip 拆** |

**判定公式**：`朗读总时长 + 单词数 × 2.5s 留白 > 15s` → 必须拆

---

## 2. 拆分原则 · 按"语义块"不按"时间"

### ❌ 错（按时间均分）

```
6 个 AT 词 = 朗读 7.5s + 留白 13s = 20.5s
按时间均分：前半 7.5s + 后半 7.5s = 15s
→ 后半还是装不下（朗读就要 3.75s + 6s 留白）
```

### ✅ 对（按语义块）

```
6 个 AT 词按概念分 3 块：
- 块 1：cat + hat + mat（"穿戴 + 场景"组）= 3 词
- 块 2：bat + fat + rat（"动作 + 反派"组）= 3 词
- 块 3：大集合（收势组）= 1 词

→ 块 1 = 12s（3 词 × 4s）
→ 块 2 = 12s（3 词 × 4s）
→ 块 3 = 6s（1 词 + 收势）
```

**为什么**：语义块 = 概念分组，听众消化有节奏感

## 3. 镜头节奏公式

| 镜头位置 | 时长 | 用途 |
|---|---|---|
| 镜头 1 | **1-2s** | 场景建立（拉远/特写）|
| 镜头 2 | **0.5-1s** | 角色跃入/动作触发 |
| 镜头 3+ | **朗读 1s + 静默 2-3s** | 每个单词一个镜头 |

**节奏样例**（Cat 4a 标版）：
- 镜头一：2s（拉远）
- 镜头二：1s（猫跃入）
- 镜头三：3s（cat · 朗读+静默）
- 镜头四：3s（hat · 朗读+静默）
- 镜头五：3s（mat · 朗读+静默）= **末帧**（不再加"收势"）

**总 12s · 节奏"2-1-3-3-3" = 短-短-长-长-长**

## 4. 末帧铁律（#36）

### 用户原话
> "把最后一个单词读完过后，仍然要像前面的每一个单词读完那样，留足够的呼吸、消化的时间，不要音频一读完，画面就结束了"

### ❌ 错（v0 思维）

```
镜头五（9-11.5s · 朗读+静默）：...
镜头六（11.5-12s · 收势）：画面定格...停留至最后一帧。
```

**问题**：末帧 = 0.5s 收势，朗读一停画面就断

### ✅ 对（v15.1）

```
镜头五（9-12s · 朗读+静默）：...（朗读 1s + 静默 2s 停留，镜头停在画面上让听众消化单词，画面最终定格在...停留至最后一帧）。
```

**核心变化**：
- 末帧 = 最后一个"朗读+静默"镜头
- 末帧静默时长 = 前面单词静默时长（2-3s）
- **没有"收势"镜头**——会让节奏断

## 5. 完整 prompt 模板（Cat 4a v3 标版）

```
主体定义：<角色 + 场景 + 卡片信息>
分镜绑定：@图N 作为唯一参考帧；
镜头一（0-2s · 场景建立）：<拉远/特写动作>，<建立性拟声>；
镜头二（2-3s · 角色跃入）：<角色跃入>，<落地拟声>；
镜头三（3-6s · 朗读+静默）：镜头定格在 <单词 1> 卡片，<拟声>一响，<角色动作>（朗读 1s + 静默 2s 停留，镜头不切不扫，停在画面上让听众消化单词）；
镜头四（6-9s · 朗读+静默）：镜头水平切到 <单词 2> 卡片，<拟声>一响，<角色动作>（朗读 1s + 静默 2s 停留，镜头停在画面上让听众消化单词）；
镜头五（9-12s · 朗读+静默）：镜头水平切到 <单词 3> 卡片，<拟声>一响，<角色动作>（朗读 1s + 静默 2s 停留，镜头停在画面上让听众消化单词，画面最终定格在<末态>，停留至最后一帧）；
参考图原有的所有文字...作为画面元素自然融入场景，模型自行设计合理的呈现节奏与动效，不重新生成任何新文字；
无任何背景音乐、无旁白人声、无哼唱。
风格：2D paper collage style, 儿童绘本纸艺拼贴风，柔和哑光色，浅米色纸张背景，柔光无强烈阴影，温馨可爱调性。
```

## 8. 反模式（不能这样写）

| 反模式 | 为什么错 |
|---|---|
| 末帧用 0.5s "收势" | 节奏断，朗读完画面跟着断 |
| 单词镜头时长 < 朗读时长 | 朗读会被截断，词听不清 |
| 单词镜头没静默 | 听众没时间消化 |
| 按时间均分拆分 | 概念被打乱，节奏感丢 |
| 6+6 词一刀切 | 段间无"呼吸"，听众累 |
