# 参考图文字元素处理（v5.0.11 末尾约束分 2 类 · 铁律 #36 修订版）

> **核心命题**（**用户原话级洞察 · 2026-06-18 奶奶绘本**）：末尾约束 `无字幕、无 Logo` 不是"翻译问题"，是"语义问题"——`无字幕` 字面意思 = "画面里不要文字" = seedance 照字面执行 = **误删参考图中作为画面装饰的文字元素**（如字母拼贴标题、品牌标识、说明句）。
>
> **修复路径** = **末尾约束按参考图是否含文字元素分流**，**不是**"逐字复制官方原话就行"。

---

## 1. 翻车时间线（奶奶 Grandma 绘本 · 3 次实战沉淀）

### 第 1 次（v5.0.6 → v5.0.7）· 默认段 4 写法
- **症状**：所有 clip 末段堆 4 句冗余（末帧定格 + 1s 动作元素 + 不得成为定格海报 + 末帧微动）
- **结果**：末帧画面静止
- **修复**：v8 简洁收尾 · 末段只写 1 句

### 第 2 次（v5.0.7 → v5.0.9）· 末尾约束从英文改回中文
- **症状**：v5.0.6 把官方"纯净无任何字幕"改写成 "No on-screen text" → 漏字 → seedance 误读成"画面里不要任何文字"
- **结果**：参考图中的画面装饰文字被误删（如字母拼贴标题）
- **修复**：v5.0.9 把英文改回中文官方原话 `无字幕、无 Logo、无水印、无背景音乐`
- **铁律 #36**（v5.0.9）：改末尾约束必逐字复制官方原话

### 第 3 次（v5.0.9 → v5.0.11）· 奶奶绘本验证发现 v5.0.9 修复**不够**
- **症状**：用户目检奶奶绘本 clip1/clip2/clip3 → 参考图彩色文字（"GRANDMA"/"奶奶"）全部被消除
- **用户原话**："**视频的质量很好，但有一个问题：参考图的彩色文字被消除了。可能是'无字幕、无 Logo'导致的**"
- **根因**：v5.0.9 错误地认为"无字幕翻车 = 翻译丢字 = 改回中文官方原话就行" = **但 `无字幕` 字面就是"不要文字"** = seedance 必照字面执行 = **跟"无字幕"字面意思冲突**
- **铁律 #36 修订**（v5.0.11）：**末尾约束按参考图是否含文字元素分流** = 分支 ① ② = `无字幕` 在分支 ② **不**写

### 隔离变量验证（v5.0.11 RP-26d 沉淀 · 2026-06-18）
- **用户原话**："**你刚刚去多做了做了两部分修改，我没有办法知道哪一个修改是有效的**"
- **clip2 v2 实战**：同时改 2 件事（去末尾约束"无字幕/无 Logo" + 加 prompt "保留 @ImageN 原有的彩色字母..."）= 文字回来了 = 不知道是哪个修改有效
- **clip3 v2 隔离验证**：单改末尾约束 = 文字**仍被删** = 确认根因 = 必须在 prompt 显式"保留 + 设计 X 动画"，仅去末尾约束不够
- **clip2 v3 完整修复**：去末尾约束"无字幕/无 Logo" + prompt 每个文字镜头段显式"保留 + 设计入场动画" = 文字完全回来 + 字母依次弹出 + 中文抖动入场 + 关键词叶子飘过特效 ✅

## 3. 文字元素识别 · image-inventory 必含清单

**Step 2 vision 阶段 = 必跑 3 步**：

### 3.1 必查项（每图必看）
1. **文字是否存在** = 看画面顶部/底部/中央/角落是否有字母拼贴、中文手写、标题、Logo、说明句
2. **文字类型** = 分类：
   - **A 类 · 彩色字母标题**（如 GRANDMA 七彩虹字母）= 高动画潜力（依次弹出入场/呼吸光晕）
   - **B 类 · 中文手写体**（如"奶奶"红色手写）= 抖动入场/翻转入场
   - **C 类 · 黑色说明句**（如 "Grandma has a nice apron."）= 淡入 + 关键词下划线飘动
   - **D 类 · 感叹号/标点**（如 "I love my Grandma!"）= 闪烁星爆
   - **E 类 · 装饰 Logo/水印** = **不**走分支 ②（用 v5.0.9 4 词约束）
3. **精确位置 + 颜色 + 字体** = 写入 image-inventory 的"文字元素详细"列

### 3.2 image-inventory 必含字段（v5.0.11 模板）

```markdown
| # | 风格 | 主体 | 背景 | 文字元素详细（颜色/字体/位置/动画潜力）| 动作 | 位置/朝向/景别 | 场景跳变 |
|---|---|---|---|---|---|---|---|
| 1 | ... | ... | ... | **底部"GRANDMA/奶奶"**——G/R/A/N/D/M/A 7 字母每个不同色（红/橙/黄/绿/青/蓝/紫）+ 中文"奶奶"红色手写体·可设计字母依次弹出入场 + 字母呼吸光晕 + 中文抖动入场 | ... | ... | ... |
```

### 3.3 文字元素分类判定（写决策表用）

| 文字类型 | 分支决策 | 原因 |
|---|---|---|
| **字母拼贴 / 中文手写 / 标题 / 装饰字 / 角字卡 / 说明句** | 走分支 ② | 是画面装饰元素，不是字幕 |
| **真实字幕条（后期添加）** | 走分支 ① | seedance 默认不加字幕 = 必删 |
| **品牌 Logo / 平台水印 / 视频角标** | 走分支 ① | 平台级 = 必删 |

## 6. 反模式（必避）

- ❌ **"无字幕、无 Logo、无水印、无背景音乐" 4 词硬套所有绘本**（v5.0.9 旧版强约束 = 参考图含文字时 = 翻车）
- ❌ **末尾约束不按参考图是否含文字元素分流**（"一套约束打天下"= 翻车）
- ❌ **vision 阶段不识别文字元素**（凭印象写 prompt = 不知道参考图含什么文字 = 误删 = 翻车）
- ❌ **prompt 不显式"保留 + 设计 X 动画"**（仅去末尾约束 ≠ 必让文字回来 = 2026-06-18 clip3 v2 隔离验证 = 文字仍被删）
- ❌ **末尾约束改 1 个变量时同时改其他变量**（隔离变量原则 = RP-26d 反模式）
