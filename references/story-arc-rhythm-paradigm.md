# 整本故事节奏范式 · Story-Arc Rhythm Paradigm

> **核心命题**：每个 clip = 整本绘本视频的一段切片，**不是**独立短片。起承转合分配在 N 个 clip 上，**不**分配在单个 clip 上。

**Skill**：picturebook-video
**沉淀日期**：2026-06-14
**触发场景**：任何绘本视频的 prompt 写作阶段（Step 5 · fill 模板前）

---

## 1. 范式对比（错 vs 对）

| 维度 | ❌ 错误范式（"单 clip 起承转合"）| ✅ 正确范式（"整本节奏分配"）|
|---|---|---|
| 起承转合归属 | 每个 clip 自己一个"起" | 整本 N 个 clip 共享 1 个"起" |
| 开篇 clip | "缓缓推近 + 轻点头 + 打量环境" = 3 镜铺垫 | 1 镜直接开 = 旁白一动 = 画面就动 |
| 中段 clip | "推近 + 切 + 跟拍" = 2-3 镜铺垫后才入情节 | 1 镜直接切入情节 = 跟着旁白走 |
| 高潮 clip | "特写推近 + 慢动作" = 反而拖慢 | 快切 + 短促 = 节奏最快 |
| 收尾 clip | "推远 + 留白静默" = 强行拖延 | 1 镜自然收 = 旁白说完画面定 |
| 默认运镜 | 推 / 拉 / 摇 / 切 至少 1 种 | **固定中景**（让参考图自身构图说话）|
| 默认动作数 | 3-4 个动作塞 1 个 clip | **1 镜 1-2 动作**（1 主 1 辅）|

**用户原话**（2026-06-14）：
> "**每一个 clip 并不是独立的，所有 clip 加起来才是一个完整的故事，每一个 clip 都是整个绘本视频的片段。不需要每一个 clip 都去构建推镜头、拉镜头，还要做很长的铺垫。**"
> "**刚开始的时候，clip1 结合参考图和旁白对应内容，可能需要做铺垫的镜头，要有起承转合的'起'这个点。有可能在后面的所有内容里，就不需要再做'起'，而是直接切入整个故事情节，节奏就很紧凑。**"
> "**后面的高潮部分可能情节发展很快，clip 设计镜头都是快切，完全有可能。也有可能直接使用参考图的镜头开始。**"
> "**本身儿童故事节奏就很快，很活泼。**"

## 2. 整本节奏分配公式（导演分镜 · 整本维度）

```
[绘本 N 个 Clip]：

Clip 1（开篇 · 唤醒）
  └─ 节奏 = 慢 · 1-2 镜铺垫 · 起 = 整本的"起"
  └─ 运镜 = 简单推近 / 固定 · 不复杂
  └─ 画面 = 介绍主体 + 环境 + 情绪定调
  └─ 例：1 鸭立草地，远景看 1.5s → 切中景停留

Clip 2..N-2（中段 · 承 + 转）
  └─ 节奏 = 快 · **1 镜 1 动作** · 直接切入情节
  └─ 运镜 = 固定 / 极简 · 不堆砌
  └─ 画面 = 参考图直接用 = 旁白说什么画面就做什么
  └─ 例：鸭子嘎嘎叫 = 1 镜固定中景 = 1 个动作完事

Clip N-1（高潮 · 转）
  └─ 节奏 = 最快 · 快切 · 多图快速叠加
  └─ 运镜 = 固定 + 直接切 = 不给铺垫
  └─ 画面 = 旁白一句 = 1 个爆发性动作

Clip N（收束 · 合）
  └─ 节奏 = 可稍慢 · 情绪留白
  └─ 运镜 = 简单（拉远 / 固定）
  └─ 画面 = 收势动作 / 1 大 1 小对视
```

**核心判断口诀**：
- clip1 = 唯一有"起"的 clip（开篇可慢）
- clip2..N = 全部"无起，直接切"
- 整本只 1 个"承转合"骨架，**不**每个 clip 各 1 个

## 3. "拟声符号" vs "持续物体" 必区分（视觉元素生命周期）

绘本参考图里**两类视觉元素**生命周期不同，prompt 处理方式完全不同：

| 元素类型 | 定义 | 视频中行为 | prompt 写法 |
|---|---|---|---|
| **拟声符号** | 漫画 / 童书传统视觉符号（声波、动作线、爆炸星、对话泡泡里的"！"等）= 表达**瞬时动作**的视觉语言 | **短暂出现后立即消失** | "3 道彩条短暂出现 0.5s 后立即消失" |
| **持续物体** | 角色、背景、文字、装饰、地标 = 场景**长期状态** | **全程保留** | "全程保留 / 不被删除或替换" |
| **收势符号** | 表示动作结束（尘埃落定、余音绕梁、标点回归）| 持续时间 = 动作结束时长 | 显式标"X 秒后归位" |

**错误反模式**（中段 clip 实际翻车）：
- 4.jpg 头部 3 道橙黄绿彩条 = **拟声符号**（"嘎嘎叫"的视觉化）
- v6 prompt 完全没有提这 3 道彩条
- 模型 = "持续发声的动作元素" = 整段 8s 始终挂着
- 翻车 = 声波永远不消失

**正确范式**：
- 写 prompt 前 = 必看 vision 参考图
- 列"叙事符号 vs 持续物体"清单（写到 image-inventory.md）
- 拟声符号 = prompt 显式说"短暂出现 X 秒后立即消失"
- 持续物体 = prompt 显式说"全程保留"

**image-inventory 必加列**（在原"有/没有"清单基础上）：

```markdown
| # | 拟声符号 | 持续物体 | 收势符号 |
|---|---------|---------|---------|
| 1 | 无 | 鸭·草地·字卡 | 无 |
| 4 | 头部 3 道橙黄绿彩条（嘎嘎叫）| 鸭·草·字卡·木牌 | 无 |
```

## 4. 整数时长 = 真实叙事秒数（不凑整）

**真实 TTS 时长计算**：
- 英文 1.0 词/秒
- 中文 3 字/秒
- 0.5s 词间停顿

**绘本节奏快 = 不要为了让"显得饱满"凑时长**：

| 旁白真实 TTS | 提交 duration | 错误做法 |
|---|---|---|
| 3-4s | 4s ✅ | 凑整 6s 或 8s = 4s 空镜 = 拖沓 |
| 5-6s | 5s 或 6s ✅ | 凑整 8s = 2-3s 空镜 = 拖沓 |
| 7-8s | 7s 或 8s ✅ | 凑整 10s = 2-3s 空镜 = 拖沓 |
| 12s | 12s ✅ | 凑整 15s = 3s 空镜 = 拖沓 |
| > 15s | 必拆 2 个 ≤ 15s | 物理上限·不凑 |

**用户原话**（2026-06-14）：
> "**鸭子在嘎嘎叫，你计算出来都只要四秒就可以完成了，为什么你要用八秒的时长来生成clip呢？**"

**判定公式**：
```
duration_seconds = TTS_real_seconds (英 1.0 词/秒 + 中 3 字/秒 + 0.5s 停顿)
duration_seconds = round(duration_seconds)  # 整数
duration_seconds = max(4, min(15, duration_seconds))  # 物理约束
# 不 + 视觉延展冗余（绘本节奏快 = 不冗余）
```

## 5. 实战自检清单（写 prompt 前必跑）

每个 clip 写 prompt 前，对照本清单 7 项：

- [ ] **1. 整本维度**：这个 clip 在整本 N 个 clip 中是开篇 / 中段 / 高潮 / 收束？节奏定位 = ?
- [ ] **2. 镜数**：开篇 1-2 镜 / 中段 1 镜 / 高潮 1 镜 / 收束 1 镜（极少 2 镜）
- [ ] **3. 运镜**：开篇可简单推近 / 中段固定 / 高潮固定 + 快切 / 收束简单
- [ ] **4. 动作数**：1-2 个核心动作（1 主 1 辅）· 不堆砌
- [ ] **5. 拟声符号**：参考图里有声波 / 动作线 / 爆炸星 → prompt 显式说"短暂出现 0.5s 后立即消失"
- [ ] **6. 持续物体**：参考图里有的角色 / 背景 / 文字 / 装饰 → prompt 显式说"全程保留"
- [ ] **7. 时长**：duration = 真实 TTS 整数秒 · 不凑整

**反模式 7 条**（必避）：

| # | 反模式 | 触发 | 修复 |
|---|---|---|---|
| 1 | 5 镜堆 8s 推来推去 | "想做得丰富点" | 1 镜 1 动作 + 固定中景 = 自然 |
| 2 | 推近 + 切中景 + 摇 | 套"标准运镜"模板 | 看整本位置 = 中段 = 固定 |
| 3 | 点头 / 摇头 / 打量 / 慢慢看 | "想加个小动作" | 中段 = 直接切 = 不铺垫 |
| 4 | 声波 / 动作线全程挂着 | 没区分"拟声符号 vs 持续物体" | 显式说"0.5s 后立即消失" |
| 5 | 旁白 4s 凑整到 8s | "让画面更饱满" | duration = TTS 真实秒数 |
| 6 | 收尾加 2s 静默 | "故事讲完留白" | 看整本位置 = 收束 = 1 镜自然收 |
| 7 | 末帧定格海报 | "想收得稳一点" | 末帧 = 1 个微动作（眨眼 / 摇尾巴 / 抬头）|

## 6. 决策树（写 prompt 时直接套）

```
写这个 clip 的 prompt 前：

Q1：这个 clip 在整本中是开篇 / 中段 / 高潮 / 收束？
   ├─ 开篇 (clip 1) → 节奏慢·1-2 镜铺垫·可简单推近
   ├─ 中段 (clip 2..N-2) → 节奏快·1 镜·固定中景·直接切入
   ├─ 高潮 (clip N-1) → 节奏最快·1 镜·快切·爆发动作
   └─ 收束 (clip N) → 节奏可慢·1 镜·自然收

Q2：参考图里有"拟声符号"（声波 / 动作线 / 爆炸星）吗？
   ├─ 有 → prompt 显式"短暂出现 0.5s 后立即消失"
   └─ 无 → 跳过这步

Q3：duration = ?
   ├─ TTS 真实秒数（英 1.0 词/秒 + 中 3 字/秒 + 0.5s 词间停顿）
   ├─ 整数（4-15 任意合法）
   └─ 不 + 视觉延展冗余（绘本节奏快）
```

## 7. 整本节奏模板（导演分镜·可直接套）

### 模板 A：4 Clip 极简绘本（总时长 15-25s）

```
Clip 1（开篇 · 6-8s）：介绍主体 + 环境 + 简单推近
Clip 2（中段 · 5-7s）：主体做动作 1 = 1 镜
Clip 3（高潮 · 4-6s）：主体做动作 2 = 1 镜 + 爆发
Clip 4（收束 · 3-5s）：自然收 = 1 镜
```

### 模板 B：5 Clip 标准绘本（总时长 30-45s）

```
Clip 1（开篇 · 6-8s）：1-2 镜铺垫
Clip 2（中段 · 5-7s）：动作 1 = 1 镜固定
Clip 3（中段 · 5-7s）：动作 2 = 1 镜固定
Clip 4（高潮 · 5-7s）：动作 3 = 1 镜快切
Clip 5（收束 · 4-6s）：自然收 = 1 镜
```

### 模板 C：领读型绘本（7 Clip · 字卡 + 故事）

```
Clip 1（开篇）：故事开篇 · 1 镜
Clip 2-6（5 个中段）：每段 1 镜 · 1 动作 · 固定
Clip 7（字卡 · 领读收束）：4 词字卡 = 4 镜 = 1 词/镜
```

## 8. 修复对照（错 → 对 · 真实案例）

| 维度 | 错（重做前）| 对（重做后）|
|---|---|---|
| 镜数 | 2 镜 | 1 镜 |
| 运镜 | 推近 + 切中景 + 摇 | 固定中景（不推不切不拉）|
| 动作 | 张嘴→合上→点头→再张→合上（4 个）| 张嘴嘎嘎叫→合上（1 个）|
| 声波 | 没提 = 持续挂着 | 显式说"短暂出现 0.5s 后立即消失" |
| 时长 | 8s（凑整）| 4s（TTS 真实 3s + 1s 视觉延展）|
| 节奏 | 慢·铺垫·呆板 | 快·紧凑·直接 |
