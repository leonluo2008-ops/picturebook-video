## 2. 4 步对账流程（铁律 `agent.任务完成度自检` 详细）

### Step 4.7.1 · ffprobe 元数据校验

**目的**：验证视频**真实身份**（时长/文件大小/分辨率）= 跟期望档位对比

```bash
ffprobe -v error -show_entries format=duration,size,bit_rate \
  -show_entries stream=width,height,codec_name \
  -of default=noprint_wrappers=1 clip7.mp4
```

**Dog 真实数据**（18:20 ffprobe 后确认）：
| 视频 | 时长 | 文件大小 | 分辨率 | 真实身份 |
|------|------|----------|--------|----------|
| clip1.mp4 | 5.08s | 1.4MB | 1280x720 | **p1 封面**（DOG!）|
| clip2.mp4 | 6.08s | 2.8MB | 1280x720 | **p2 草地**（happy dog）|
| clip3.mp4 | 6.08s | 2.7MB | 1280x720 | **p3 飞奔**（runs fast）|
| clip7.mp4 | 14.09s | 5.6MB | 1280x720 | **p7 OG 家族**（5 词押韵）✓ |

**判断标准**：
- ✅ duration ≈ 期望档位（5s/6s/7s/8s/12s/14s · ±0.2s 内）
- ✅ file size 跟时长成正比（5s ≈ 1.4MB / 6s ≈ 2.7-2.8MB / 14s ≈ 5.4MB）
- ✅ 分辨率 1280x720（16:9 720P）
- ❌ **任何一项不对 = 任务失败 + 立即停止汇报**

### Step 4.7.2 · 抽帧 + native vision 视觉目检

**目的**：验证视频**视觉内容** = 跟原图期望对比

```bash
# 抽帧（每秒 1 帧）
ffmpeg -y -i clip7.mp4 -vf "fps=1" clip7-frame-%02d.png

# native vision 看 3 张关键帧（首帧/中段/末帧）
# 关键：用 native vision（直接喂给当前 model），**不**用 mcp_zai_analyze_image（转述丢像素）
```

**Dog clip7 真实视觉目检**（native vision 后确认）：
- ✅ **首帧（frame-01）**：完整 5 词卡片墙（dog/log/frog/jog/fog + 1 装饰格）= **100% 跟原图 7.jpg 一致**
- ✅ **中段（frame-07）**：jog 单卡片特写 = 镜头切换节奏正确
- ✅ **末帧（frame-14）**：回归全景 = 跟原图 6 格布局完全一致
- **真测结论**：v22 修复成功 = 真实

**判断标准**：
- ✅ 关键视觉元素**跟原图一致**（5 词卡片墙 = 5 词卡片墙 · 不允许"扩写"或"偏离"）
- ✅ 镜头切换节奏**跟 prompt 描述一致**（5 词 5 镜 + 1 静默沉淀）
- ❌ 视觉内容**跟原图不符** = 翻车 = 立即停止汇报 + 重发 C 改 prompt 重跑

### Step 4.7.3 · 校验文件名 vs 内容身份一致

**目的**：**最致命的一步**——**避免错位发视频**

```python
# 校验逻辑
expected = {
    "clip1.mp4": "p1 封面 Eric Carle 拼贴风 + 棕白小狗 + DOG/小狗 文字",
    "clip2.mp4": "p2 草地小狗欢快 + A happy dog",
    "clip3.mp4": "p3 飞奔 + 速度线 + 风扬耳朵/尾巴 + The dog runs fast",
    "clip7.mp4": "p7 OG 家族 5 词卡片墙 (dog/log/frog/jog/fog)"
}
# 对每个 clip：vision 看到的内容 = expected[clip_name] 才算通过
```

**Dog 18:15 事故现场**：
- 我汇报"clip7 视觉目检通过"时，**发的就是 clip1/2/3 的帧**
- **没校验** "clip7.mp4 这个文件里的内容是不是 p7 OG 家族"
- **根因**：**没做文件名 vs 内容身份一致性校验**

**判断口诀**：
- ✅ clip7.mp4 的内容 = p7 OG 家族
- ❌ clip7.mp4 的内容 = p1 封面 = **错位 = 立即停止汇报**
