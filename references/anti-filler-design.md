# 反凑数设计（anti-filler-design）

> **元命题**：静默/情节填充 = 编排工具不是义务 · 时长 = 用户给 TTS 优先
>
> **来源**：2026-06-16 用户原话级纠错 · 升铁律 #27

---

## 1 · 核心命题

```
静默 = 编排工具不是义务
情节填充 = 不必
时长 = 用户给 TTS 优先
   ↓
"够用就好 = 不为饱满凑数"
   ↓
旁白 4s = 4s（不凑 8s）
旁白 6s = 6s（不凑 10s）
整本 N 个 2s 静默叠加 = 节奏断裂（必删）
```

---

## 2 · 静默决策树

```
问：这段要不要末帧静默？
   ↓
① 故事连贯 = 静默？→ NO（直接切下一个 Clip）
② 视觉断点 = 静默？→ YES（短静默 0.5-1s 即可）
③ 凭印象"末帧要消化" = 静默？→ NO（**不**为消化硬塞）
   ↓
判定：静默 = 编排工具 = 按故事需要决定
   ↓
**不**默认 ≥ 2s 末帧静默
**不**每段都塞静默
**不**为"显得饱满"加静默
```

**反模式**：
- ❌ 每段硬塞 ≥ 2s 末帧静默
- ❌ 凭印象套 v5 公式"末帧静默 2.9-3.8s"
- ❌ 整本 N 个 2s 静默叠加 = 节奏断裂
- ❌ 视觉断点场景**不**加静默 = 显得"没消化"（错）· **不**加 = OK

---

## 3 · 情节填充决策树

```
问：这段要不要加铺垫/转折/收势？
   ↓
① 旁白在说什么？→ 镜头跟着旁白走
② 旁白在"亮相" = 镜头"亮相"（直接展示）
③ 旁白在"特征" = 镜头"特征"（直接展示）
④ 旁白在"高潮" = 镜头"高潮"（直接展示）
⑤ 旁白在"收势" = 镜头"收势"（直接展示）
   ↓
判定：情节 = 旁白在讲什么 = 镜头在展示什么
   ↓
**不**为"显得有起承转合"加铺垫/转折/收势
**不**为"镜头层次感"加"推近→切中景→特写"
**不**凭印象加 4 步铺垫（"缓缓推近→点头→再张开→收势"）
```

**反模式**：
- ❌ 凭印象加"缓缓推近→切中景→点头→再张开"4 步铺垫
- ❌ 默认"开篇 = 慢铺"（短单词开篇 = 中段节奏 = 直接切）
- ❌ 整本每个 Clip 都做"起承转合"
- ❌ 旁白短 = 加多余动作"显得饱满"

## 4 · 时长决策树

```
问：视频总时长 = 多少？
   ↓
① 用户给 TTS 数值？
   ├─ 是 → 用户说"严格匹配 TTS" / "TTS 有冗余" / 视频总时长 = TTS？
   │        ├─ 是 → 视频总时长 = 用户给 TTS（**不**加 5s 冗余）
   │        └─ 否（默认）→ 视频总时长 = 用户给 TTS + 5s 冗余
   └─ 否 → 兜底公式 1.0 词/秒 + 3 字/秒 + 0.5s 词间停顿
   ↓
问：每 Clip 时长 = 多少？
   ↓
② 该 Clip 段 TTS 真实秒数（必算 + 必跟用户给 TTS 校对）
   ↓
③ 整数化（向上取整 + 0.5s 末帧微动空间）
   ↓
④ 物理约束 ≤ 15s（> 15s 必拆）
   ↓
⑤ **不**为了"显得时长饱满"额外凑整
   ↓
判定：时长 = 真实叙事秒数 + 必要冗余
```

**判定口诀**：
- **够用就好** = 旁白 4s = 4s，不凑 8s
- **不凑数** = 真实 TTS 6s = 6s，不为了"显得饱满"凑到 10s
- **末帧微动** = 必查（铁律 #15 原文"末帧定格前持续微动不得成为定格海报"）= **不**为了"末帧有动作"硬加多余动作

## 5 · 3 类常见"为饱满而填"动作（识别 + 修复）

### 类型 1 · 末帧静默 ≥ 2s（"消化时间"思维）

**症状**：每段末尾加 2-3s 静默让旁白"消化"
**根因**：默认"短句需要消化时间"
**判定**：绘本短句 = 旁白已经够短 = **不**需要"消化"
**修复**：视觉断点 = 短静默 0.5-1s / 故事连贯 = 0 静默
**反例**：整本 N 个 2s 静默叠加 = 节奏断裂

### 类型 2 · 镜头内"推近→切→特写"3 步铺垫（"镜头层次"思维）

**症状**：1 个 Clip 内塞"缓缓推近→切中景→特写"3 步铺垫
**根因**：默认"动态 = 镜头运动"
**判定**：参考图驱动 = 固定原景别 = **不**推不切不拉
**修复**：运镜 = 固定原景别 + 主体动作循环（不切新景别）
**反例**：3 步铺垫 + 4 步动作 = 8 个动作里必有 2 个跟参考图姿态冲突

### 类型 3 · 末帧定格海报（"末帧要有动作"思维）

**症状**：末帧描述"画面定格/海报式/留白"等
**根因**：默认"末帧 = 静态"
**判定**：末帧微动 = 必查（持续微动）
**修复**：末帧微动 = 参考图原姿态的微动（不脱离参考图）
**反例**：3 个镜头共用"长耳朵轻轻摆动" = 没有"末帧差异" = 没有"故事推进"

## 6 · 实战案例（2026-06-16 Run 跑）

### 案例 A · 真实 TTS 4s vs 拟时长 4s（不凑数）

```
段 1 "RUN! 跑！RUN！" 真实 TTS = 1.71s
   ↓
按真实 TTS 算 = 2s（向上取整）
   ↓
**不**为了"显得饱满"凑到 4s/5s
   ↓
但每 Clip 拟时长要装下"段 + 末帧微动空间" = 0.5s 缓冲
   ↓
最后拟时长 = 真实 TTS + 0.5s 末帧微动 = 向上取整
```

### 案例 B · 4 个 Clip 时长分配（不硬塞静默）

```
Clip 1 (段 1+2+3)  真实 TTS 9.00s  拟时长 9s  = 0 静默
Clip 2 (段 4+5)    真实 TTS 9.54s  拟时长 10s = 0 静默
Clip 3 (段 6)      真实 TTS 4.86s  拟时长 5s  = 0 静默
Clip 4 (段 7+8)    真实 TTS 7.07s  拟时长 8s  = 0 静默
视频总时长 = 32s（用户给 28s + 5s 冗余 = 33s 目标 = 差 -1s 在容忍内）
   ↓
**不**为"显得饱满"硬塞 2s 末帧静默
**不**凭印象"整本 N 个 2s 静默"叠加
```

## 7 · 自检清单（写 prompt 前必跑）

- [ ] **Q1 末帧静默 = 视觉断点需要？**（不是默认 ≥ 2s）
- [ ] **Q2 镜头序列 = 跟着旁白走？**（不是凭印象加铺垫/转折/收势）
- [ ] **Q3 视频总时长 = 用户给 TTS 优先？**（不加 5s 除非默认）
- [ ] **Q4 每 Clip 拟时长 = 真实 TTS + 0.5s 末帧微动？**（不为了"显得饱满"凑整）
- [ ] **Q5 拟时长总和 = 用户给 TTS ± 5s 容忍？**（不超额也不偷工）
- [ ] **Q6 整本 N 个 Clip 静默叠加 = 节奏没断？**（不硬塞 N 个 2s）
- [ ] **Q7 末帧微动 = 参考图原姿态的微动？**（不脱离参考图）
- [ ] **Q8 镜头内 3 步铺垫 = 没有？**（短句 = 直接切）
- [ ] **Q9 真实 TTS 4s = 没凑到 8s？**（不为了"显得时长饱满"）
- [ ] **Q10 末帧微动 = 跟其他镜头有差异？**（不共用"轻轻摆动"）

**9/10 全过 = 写完 · 任 1 项失败 = 必修**

## 8 · 跟现有铁律的配合

| 铁律 | 配合点 |
|---|---|
| **#6** 静默 = 编排工具不是铁律 | **强化** = 不硬塞 ≥ 2s |
| **#7** 视频总时长 = 用户给 TTS + 5s 冗余 | **强化** = 用户说"严格匹配"不加 5s |
| **#11** 整本节奏 = N 个 clip 节奏分配 | **强化** = 不每个 clip 各做各的起承转合 |
| **#12** 时长 = 真实叙事秒数不凑整 | **强化** = 4s 旁白 = 4s 不凑 8s |
| **#15** 1 镜 1 种运镜 + 多动作 OK | **强化** = 镜头内 3 步铺垫 = 反模式 |
| **#26** 参考图驱动设计铁律（本轮新增）| **配合** = 末帧微动 = 参考图原姿态的微动 |

## 9 · 跨场景通用

- ✅ 绘本视频（旁白 + 图）= 本场景
- ✅ 漫剧分镜（对白 + 角色图）
- ✅ AI 短剧（旁白 + 场景图）
- ✅ 短视频（旁白 + 素材图）
- ✅ 教学视频/产品演示
- ✅ 任何"声音 + 视觉"组合的创作场景

**通用命题**：**够用就好** = 不为饱满凑数 · 静默/情节/时长 = 按真实需要定
