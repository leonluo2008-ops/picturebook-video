## 📖 三份文档定位

### 1-系列教程（入门 + 功能总览）
- 适合：刚接触 seedance，想知道它能做什么 / 怎么用
- 内容：模型介绍、基本参数、功能列表
- 用法：**先用这份建立全局认知**

### 2-系列提示词指南（写 prompt 必查）
- 适合：写 prompt 时不确定怎么表达
- 内容：提示词结构、风格词库、镜头语言、运镜术语
- 用法：**写 v15 模板之前先扫一遍术语表**

### 3-视频生成教程（端到端流程）
- 适合：做完整视频项目（多镜头/转场/角色一致性）
- 内容：端到端流程、场景设计、角色锁定、参考图用法
- 用法：**做漫剧/多角色视频时必看**

## ⚠️ 注意事项

1. **官方文档是兜底，不是首选**——picturebook-video skill 的 v15 模板 + 4 维控制原理是经过 pic1-4 实战验证的，**比官方文档更具体更落地**
2. **官方文档可能过期**——seedance 模型会迭代，文档是 2026-06 拿的快照，新功能可能没收录
3. **prompt 写法有 2 个权威源**：
   - **本 skill 的 v15 模板 + v6 升级**（实战沉淀）
   - **官方 2-系列提示词指南**（术语参考）
   - 两个冲突时**以 v15 模板为准**（实战 > 理论）
4. **不要把官方文档内容直接复制进 prompt**——先理解原理，再用 v15 4 段结构改写
