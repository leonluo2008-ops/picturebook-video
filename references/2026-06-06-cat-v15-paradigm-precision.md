# Cat v15 范式精准 4 点（铁律 #53）

> **核心洞察**（用户 2026-06-06 原话）："我说的是对画面的镜头控制，画面的内容控制，特别是对clip做拆分的时候，那种控制，非常精准。"
>
> **Pic2 v1.0.0 实战发现**：C 子 agent 缺 Cat v15 范式 4 点精准度，**画面在配音**（"嘴巴半张开说 Please"），不是**画面先讲语义**。
>
> **Cat 6 Clip 实战**（2026-06-05 v15.1 范式跑通）：拆 Clip = 语义块（pat/cat/mat/sat/rat/fat 6 个独立画面场景），不是切同一图 6 段。

## 4 点精准

### ① 拆 Clip 维度 = **语义块**（不是时间块）

**Cat 范式**：1 张图 + 6 个 AT 韵词 + 14s 朗读 → 拆成 6 个 Clip（每 Clip 1 韵词 + 2.3s 朗读 + 完整画面场景）

**反模式**：v0.7.1+pic7 时代拆 Clip = "前 2.5s 一段 + 后 2.5s 一段" = **切同一图 2 段** = 画面重复

**Pic2 应用**：1 张图 = 1 句话 = 1 Clip = 1 个完整画面场景（Mama 翻开书页 / Papa 张开双臂 / Teacher 翻开乐谱 / ...）

### ② 拟声 = **故事动作音**（不是装饰音）

**Cat 范式**：
- `pat 一响` = 猫掌落地（pat 韵词 + 掌按地动作）
- `cat 一响` = 小猫抬头（cat 韵词 + 抬头动作）
- `mat 一响` = 垫子滑出（mat 韵词 + 垫子出现）
- `沙沙` = 窗帘摆动（拟声 + 物体动作）

**反模式**：v0.7.1+pic7 时代拟声 = "沙沙/叮/咚/叮咚" 4 个装饰音随机分配 = **跟画面脱钩**

**Pic2 应用**：
- 镜头二拟声 = clip_narrative 故事动作的第 2 拍发出的音
- 例：Clip 1 镜头二"妈妈翻动书页" → 拟声 `唰`
- 例：Clip 2 镜头二"爸爸张开双臂" → 拟声 `噗`

### ③ 画面 vs 朗读 = **画面先讲语义，朗读强化读音**

**Cat 范式**：
- pat Clip 画面 = 猫掌按地动作先呈现（**讲完 pat 的故事**）
- 朗读 pat 强化读音（**让小朋友听清 pat 怎么读**）
- 顺序 = 动作 → 词（不是"嘴巴半张开说 pat"）

**反模式**：v0.7.1+pic7 时代 prompt 写"小兔子嘴巴半张开做说话口型礼貌地说出 Please" = **画面在配音** = 绘本调性温柔内敛时角色不会张嘴

**Pic2 应用**（clip_narrative 必填）：
- Clip 1 = 妈妈翻动书页 + 小兔子眼睛盯着书页插图（**画面先讲"请读书"的故事**）
- Clip 5 = 兔子站在玩具房门口探身（**画面先讲"请问我可以玩吗"的故事**）
- 朗读 Please **强化**读音（**不是嘴巴半张开**）

### ④ 末帧 = **本 Clip 故事动作停留 + 目标词重读窗口 + 画面微动**

**Cat 范式**：
- pat Clip 末帧 = 猫掌按在地上不动（**pat 动作停留**）+ 静默 2s（**pat 重读窗口**）+ 画面微动（猫掌压力地）
- 末帧给小朋友"消化韵词"的时间（**不是单纯"收势"**）

**反模式**：v0.7.1+pic7 时代末帧 = "画面最终定格在温馨画面" = **定格海报** = 画面死去 4-5s

**Pic2 应用**：
- Clip 1 末帧 = 妈妈翻开的书页 + 小兔子仰头盯书（**故事动作停留**）+ 目标词 Please 重读窗口（**让小朋友学发音**）+ 画面微动（小兔子耳朵轻颤 0.2s + 妈妈手轻抚书页 0.2s + 橙色窗帘摆动 0.3s）

## Cat v15 范式精准 4 点 vs v0.7.1+pic7 反模式

| 维度 | Cat v15 范式 | v0.7.1+pic7 反模式 | 修复方向 |
|---|---|---|---|
| 拆 Clip | 语义块（1 韵词 1 画面）| 时间块（前 2.5s + 后 2.5s）| 1 张图 1 句旁白 1 Clip |
| 拟声 | 故事动作音（pat 一响 = 猫掌落地）| 装饰音（沙沙/叮/咚/叮咚 随机）| 拟声 = clip_narrative 故事动作音 |
