## 3. 阶段 4 · 兜底文档标准化

> **任何 skill 仓"参考"性质的文档**（官方教程 / 知识库 / 长期参考资料）都应按这个模式落地。

**位置**：`references/<文档类别>/`

**强制文件**：
1. **N 文件本体**（如 6 docx/md）—— 原始资料
2. **`README.md`**（必有）—— 索引文件，内容：
   - 文件位置（树状结构）
   - 何时查哪份（场景对照表）
   - 文档定位
   - 兜底决策树（skill references → 官方 docs → web_search）
   - 注意事项（不要直接复制官方内容等）

**SKILL.md 必加引用**：失败模式决策树里加一行"遇到 skill 兜不住的问题 → 先查 [references/.../README.md]"

**本次实战**：`references/seedance-official-docs/`（508K · 6 文件 + 1 README）
