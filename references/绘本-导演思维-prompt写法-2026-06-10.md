## 1. 4 步原则详解

### Step 1 · 以参考图作为首帧推断后面 clip 时长

**❌ 反模式**：套"5s=2-3 镜头 / 8s=3-5 镜头"等固定公式
**✅ 正解**：先看 p1 实际画面 → 推断"仓鼠可能接下来钻进木屑 / 跑上转轮 / 啃种子" → 根据动作复杂度定 5/6/7s

**实操步骤**：
1. vision_analyze 抽 t=0.5s 帧（**已含场景对位检查**）
2. 读出 4 个关键事实：仓鼠在哪 / 做什么 / 旁边有什么道具 / 文字位置
3. 推断"旁白关键词 → 仓鼠在做什么动作"（"a cute"= 圆滚可爱 / "runs in wheel"= 跑轮）
4. 动作复杂度 → 时长（1 个动作 = 5-6s / 1 个动作 + 静默 = 6-7s / 收势 = 7s）

### Step 2 · 结合旁白内容推断接下来可能发生的事情

**❌ 反模式**：看到 "happy hamster eats" = 写"仓鼠笑 + 仓鼠吃"（文字表面）
**✅ 正解**：看参考图实际场景（草地 + 木碟 + 葵花籽）→ 推断"眯眼笑 + 双手捧瓜子 + 木碟备用" → 把画面已有元素**当故事起点**写 prompt

**双源推断口诀**：
- 旁白提供"动作关键词"（cute / fat / happy / runs / find）
- 参考图提供"动作场景"（草地/木屑/跑轮/木屋/碟子）
- 两源交集 = 真实动作（happy + 草地 + 碟子 = "在草地上开心地吃"）

### Step 3 · 根据故事情节设计镜头 · 不按固定模式

**❌ 反模式**：v15 范式被理解为"6 段骨架 = 6 段文字" + 每段固定"3 镜头 = 建立/单词/收势"
**✅ 正解**：镜头数 = 动作步骤数（钻木屑 = 1 个钻的动作镜头 / 跑轮 = 1 个跑的动作镜头 / 探头 = 1 个探头的动作镜头），**不**为凑数加空镜

**每 Clip 镜头数 1-3 个**（不固定 3 镜）：
| 动作步骤数 | 镜头数 | 末帧 |
|---|---|---|
| 1 个动作（钻木屑/跑轮/探头）| 1-2 镜 | 1 镜真正静默 |
| 1 个动作 + 静默消化 | 2-3 镜 | 末镜真正静默 |
| 多个动作 | 3 镜 | 末镜真正静默 |

### Step 4 · 留出呼吸感的静默内容

**❌ 反模式**：末帧写"画面继续微动"（v15 旧版踩过 = 实际不微动）
**✅ 正解**：末帧镜头**写"画面不再动 · 整段画面留出静默 · 让 XX 在小朋友心中沉淀"**—— 真正的静默（不是"微动"）

**静默时长 = 整段时长的 30-40%**：
- 5s 留 1.5-2s 静默
- 6s 留 2-2.5s 静默
- 7s 留 2-2.5s 静默

**末帧 prompt 写法模板**：
```
镜头 N（消化静默·X-Ys）：缓推回中景，画面不再动。<主角小仓鼠>保持 XX 姿势，圆眼睛安静望向 XX，撞色文字"hamster"在画面 XX 方静立。整段画面留出静默，让"XX"这个画面在小朋友心中沉淀。
```

## 5. "首帧" ≠ "首尾帧" 关键澄清

| 维度 | 4 步原则的"首帧" | seedance "首尾帧范式" |
|---|---|---|
| **含义** | 视觉设计起点 = 参考图 | API 参数 = `--image` + `--last-frame` 锁住首/尾两帧 |
| **作用** | 推断情节 + 决定镜头内容 | 锁帧 = 模型只能在两帧之间插值 |
| **是否禁用** | ✅ 必用 | ❌ 禁用（v1.2.0+pic21 红线）|
| **使用方式** | "看 p1 仓鼠捧种子 → 推断 p2 仓鼠在户外草地" | "锁 p1 仓鼠 + 锁 p2 仓鼠 → 模型在中间过渡" |

**判断口诀**：
- 看到"首帧" = 思路起点 = "以参考图为基准来设计" ✅
- 看到"首尾帧" = seedance 参数 = `--image`+`--last-frame` ❌ 禁用
