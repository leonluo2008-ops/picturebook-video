## Phase 5 启动前必读三铁律（2026-06-05 Say 说绘本踩坑沉淀）

| 铁律 | 核心 | 何时触发 | 详情 |
|------|------|---------|------|
| **#25 旁白语言版本** | 用户说"X 版旁白"= X 版**完整那一行**（不是"删 Y 留 X"）| 双栏素材表（中英对照）/ 用户指定语言版本 | SKILL.md 关键教训 #25 |
| **#26 彩色文字自然融入** | 末帧文字消失 = 合规；参考图所有文字至少出现一次 = 硬约束 | 单测门报"末帧文字丢失"时**先判断是否合规** | SKILL.md 关键教训 #26 |
| **#27 运镜服务于故事** | 按画面类型定制（封面/动作/多人/情感/礼物/动态），不套固定公式 | 写 prompt 选运镜时 | SKILL.md 关键教训 #27 |

## 范式选择决策

1. **绘本多图** → v14（默认）
2. **绘本单图但多 Clip 想统一 BGM** → v10
3. **单 Clip 需精准动作** → v6/v7
4. **任何场景别选** → v3 / v12
