## [0.6.0] - 2026-06-05

### 新增（v15.1 · 长旁白单图多 Clip 拆分规范）
- **铁律 #36 末帧 = 朗读+静默**：末帧镜头时长 = 朗读 1s + 静默 2-3s；不要"0.5s 收势"标签
- **铁律 #37 长旁白单图多 Clip 拆分 = 语义块**：按概念分块（3+3+1）不按时间均分（7.5+7.5）
- **references/长旁白拆分规范-v15.1.md**：5.3KB 详细文档（触发条件/语义块原则/镜头节奏公式/末帧铁律/Cat 4a v3 标版 prompt 模板/反模式）
- **VERSION_INDEX**：v15.1 行 · "Cat 4a v3 标版 · 单图朗读 + 留白 > 15s 上限时必走"

### v15.1 核心 4 件
1. **触发公式**：`朗读总时长 + 单词数 × 2.5s 留白 > 15s` → 必须拆
2. **语义块拆分**（不按时间均分）：3+3+1 比 7.5+7.5 节奏感强
3. **节奏公式**：动作-动作-词-词-词（"2-1-3-3-3" 标版 = 短-短-长-长-长）
4. **末帧铁律**：用户原话 "把最后一个单词读完过后，仍然要像前面的每一个单词读完那样，留足够的呼吸、消化的时间"

### Cat 4a v3 标版实测数据
- Task ID: cgt-20260605190341-drj78
- 12s · 16:9 · 信息密度 25%（3 词 / 12s）
- Token: 260,100
- 用户评价: "画面效果、镜头设计、领读的呼吸感是够的" ✅

### 状态
- ✅ v15.1 = 长旁白拆分的标版规范（绘本默认子范式）
- ⏳ v15.2 候选 = 单图多 Clip 衔接（`--last-frame` 锁末帧 + 拟声渐弱 + 情绪承接）—— 待 4a→4b 实测
- v14 仍可用（备选，4 段骨架 + v7 静默型）
